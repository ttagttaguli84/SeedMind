# 채집 시스템 (Gathering System) 상세 설계

> 작성: Claude Code (Opus 4.6) | 2026-04-07

## Context

이 문서는 SeedMind의 채집 시스템을 상세히 기술한다. 채집은 농장 주변 자연 환경에서 야생 식물, 버섯, 광물 등을 수집하는 **보조 활동**으로, 경작과 낚시에 이은 세 번째 수입원이자 핵심 재료 공급 경로이다.

**설계 목표**: 채집은 낚시의 "집중 + 미니게임" 성격과 대비되는 **"탐색 + 수집"** 성격의 활동이다. 매일 채집 포인트를 순회하며 계절마다 달라지는 자연물을 발견하고, 경작/낚시 재료로 활용하거나 판매하는 구조다. 채집만으로 경작을 대체할 수 있는 수준의 수익을 내서는 안 되며, "보조 수입 + 도구/낚싯대 업그레이드 재료 공급 + 가공 소재 + 계절 감각 강화" 역할에 머물러야 한다.

**낚시 시스템과의 차별화**:

| 관점 | 낚시 | 채집 |
|------|------|------|
| 인터랙션 | 미니게임 (집중, 반응 속도) | 이동 + 클릭 (탐색, 경로 최적화) |
| 소요 시간 | 1회 15~30초 (게임 내), 고정 위치 | 순회 5~10분 (게임 내), 이동 필요 |
| 리소스 비용 | 에너지 2~3/회 | 에너지 0~1/회 (맨손 0, 도구 1) |
| 출현 패턴 | 실시간 확률 판정 | 일 단위 리스폰, 고정 포인트 |
| 핵심 재미 | 긴장감, 대물 기대감 | 발견의 기쁨, 루트 최적화 |

## 1. 개요

### 1.1 채집 활동의 핵심 루프

```
[채집 포인트 확인] → [이동] → [채집 (클릭)] → [아이템 획득]
                                                    ↓
                                         [판매 / 가공 재료 / 업그레이드 재료]
```

### 1.2 해금 조건

| 파라미터 | 값 | 비고 |
|----------|-----|------|
| 기본 채집 | 게임 시작 시 | Zone A 잡초 제거 시 일부 채집물 드롭 |
| 본격 채집 | Zone D 해금 (레벨 5, 2,500G) | 숲 가장자리 채집 포인트 8개소 활성화 (-> see `docs/systems/farm-expansion.md` 섹션 4.1) |
| 동굴 입구 채집 | Zone D 해금 후 자동 | 동굴 입구(Zone D 동쪽 끝) 광물 채집 포인트 3개소 |

**설계 의도**: 기본 채집(잡초 제거 드롭)은 게임 시작부터 자연스럽게 노출되어 "채집이라는 활동이 있다"는 것을 학습시킨다. Zone D 해금 이후 본격적인 채집 루프가 시작되며, 이 시점은 낚시(Zone F, 레벨 5)와 비슷한 시기이므로 플레이어에게 "낚시 vs 채집" 시간 배분 선택을 유도한다.

### 1.3 채집 인터랙션

```
[플레이어] 채집 포인트 인접 타일에서
    |
    ├─ [맨손] E키 → 수집 애니메이션 (1초) → 아이템 획득
    |   에너지 소모: 0
    |   품질 보정: 없음
    |
    └─ [채집 낫 장착] E키 → 빠른 수집 애니메이션 (0.5초) → 아이템 획득
        에너지 소모: 1
        품질 보정: 도구 등급에 따라 (섹션 5 참조)
```

**설계 의도**: 맨손으로도 채집이 가능하여 진입 장벽을 낮춘다. 도구를 사용하면 속도가 빨라지고 품질 보정이 붙어, 도구 투자에 대한 보상이 존재한다. 에너지 소모가 극히 낮아(0~1) "빈 시간에 가볍게 돌아보는" 활동으로 포지셔닝한다.

### 2.2 리스폰 메카닉

| 파라미터 | 값 | 비고 |
|----------|-----|------|
| 리스폰 주기 (식물/버섯/열매) | 2일 | 06:00(Dawn) 배치 처리 시 판정 (-> see `docs/systems/time-season.md` 섹션 1.2) |

| 계절 전환 시 | 즉시 리스폰 | 새 계절 Day 1에 모든 포인트 100% 리스폰 (신규 계절 아이템) |

| 최대 미수확 누적 | 1개 | 수확하지 않으면 다음 리스폰 안 됨 (아이템 방치 페널티) |

**일일 채집 가능 수량**: 22개소 중 평균 80% 리스폰 = 약 17~18개 아이템/일. 실제로 전 구역을 순회하려면 Zone D/E/F를 모두 해금해야 하므로, 초기에는 Zone D의 15개소(12~13개/일)만 접근 가능.

### 2.3 시각적 표현

| 상태 | 시각 | 비고 |
|------|------|------|
| 채집 가능 | 해당 자연물이 보임 (버섯, 꽃, 광석 결정 등) + 미세 반짝임 파티클 | 발견 용이성 확보 |

### 3.8 DES-015 연계: 광석 획득 경로 확정

이 설계로 fishing-system.md 섹션 1.1의 [OPEN] 항목("강화/전설 낚싯대의 재료(구리 광석, 금 광석) 획득 경로")을 해소한다.

**강화 낚싯대 재료: 구리 광석 x5**
- 동굴 입구 3개소, 3일 리스폰, Uncommon 가중치 30/100
- 3일마다 평균 3 x 0.8(리스폰) x 0.30(Uncommon중 구리 확률) = ~0.72개/3일 = ~0.24개/일
- 5개 수집 예상 소요: 약 20~25일 (3주)
- **대안 경로**: 여행 상인(-> see `docs/content/npcs.md` 섹션 6.3)에게서 구매 가능 (섹션 7 참조)

**전설 낚싯대 재료: 금 광석 x3**
- 동굴 입구 3개소, 3일 리스폰, Rare 가중치 13/100 (금 광석은 Rare 중 50% 확률)
- 3일마다 평균 3 x 0.8 x 0.065 = ~0.156개/3일 = ~0.05개/일
- 3개 수집 예상 소요: 약 55~60일 (2계절)
- **대안 경로**: 여행 상인 구매 또는 낚시 보물 상자(-> see `docs/systems/fishing-system.md` 섹션 6.4)

**설계 의도**: 구리 광석은 "채집을 꾸준히 하면 1계절 내 충분히 모을 수 있는" 접근성 있는 재료. 금 광석은 "채집만으로는 2계절이 걸리므로 여행 상인/보물 상자를 병행하는 것이 효율적"인 희소 재료. 이를 통해 채집-낚시-상인 간의 교차 의존성을 만든다.

## 5. 도구 연계

### 5.1 설계 결정: 채집 낫 (Gathering Sickle)

**채택**: 채집 전용 도구 — 채집 낫 (Gathering Sickle)
**불채택 대안들**:
- 기존 농업 낫(Sickle) 공유: 낫은 수확용이며, 채집 숙련도와 분리된 성장 트랙에 도구를 공유하면 혼란 유발
- 도구 불필요 (맨손 전용): 성장감 부족, 투자처 부재
- 바구니(Basket) 도구: 직관적이지만 "수확"과 시각적으로 차별화 어려움

**채택 근거**:
- 낚싯대가 낚시 전용이듯, 채집 낫은 채집 전용으로 "보조 활동에 전용 도구가 있다"는 일관된 패턴
- 대장간 업그레이드 경로에 자연스럽게 편입 (기존 3등급 체계 활용)
- 에너지 1 소모 대신 속도 + 품질 보정이라는 트레이드오프 제공

### 5.2 채집 낫 등급

| 등급 | 영문 ID | 획득 방법 | 해금 조건 | 비용 | 효과 |
|------|---------|----------|----------|------|------|
| 기본 채집 낫 | `sickle_gather_basic` | Zone D 해금 시 대장간에서 구매 | Zone D 해금 | 200G | 채집 속도 x0.5 (1초 -> 0.5초), Silver 품질 기회 |
| 강화 채집 낫 | `sickle_gather_reinforced` | 대장간 업그레이드 | 채집 숙련도 Lv.3 | 1,000G + 구리 광석 x3 | 채집 속도 x0.5, Gold 품질 기회 |
| 전설 채집 낫 | `sickle_gather_legendary` | 대장간 업그레이드 | 채집 숙련도 Lv.7 | 3,000G + 금 광석 x2 + 수정 원석 x1 | 채집 속도 x0.5, Iridium 품질 기회, 더블 드롭 +5% |

**설계 의도**: 도구 업그레이드 시스템(-> see `docs/systems/tool-upgrade.md`)과 동일한 3단계 구조. 기본 채집 낫은 Zone D 해금 후 즉시 구매할 수 있는 저렴한 가격(200G)으로 설정하여 "채집 시스템에 투자할지 말지"의 가벼운 결정을 유도한다.

**재료 경로의 순환 구조**: 구리 광석은 채집으로 얻고, 이를 채집 낫 강화에도 사용한다. 이는 "채집 → 광석 획득 → 도구 강화 → 더 나은 채집" 순환 루프를 형성한다. 동시에 낚싯대 강화(구리 광석 x5)와 경쟁하므로 "채집 도구 vs 낚시 도구" 투자 우선순위 결정이 필요하다.

### 5.3 채집 낫 인벤토리 규칙

- 채집 낫은 도구 슬롯에 장착 (-> see `docs/systems/inventory-system.md`)
- 스택 불가 (도구와 동일)
- 업그레이드 시 기존 등급 소멸, 상위 등급으로 교체
- 업그레이드 소요 시간: Reinforced 1일, Legendary 2일 (도구 업그레이드와 동일, -> see `docs/systems/tool-upgrade.md` 섹션 2.1)
- 업그레이드 중 맨손 채집은 여전히 가능 (낚싯대와 달리 맨손 대안이 있으므로 도구 부재 페널티 완화)

## 6. 경제적 포지션

### 6.1 수입 포지셔닝

**핵심 원칙**: 채집은 "가장 낮은 노력 대비 적정 수입" 포지션이다. 에너지 소모가 거의 없고 미니게임도 없어, 시간(이동)만 투자하는 대신 수입도 낚시/경작보다 낮다.

**채집 수익 시뮬레이션** (숙련도 Lv.1, 맨손 기준, BAL-016 판매가 40% 하향 적용):

```
Zone D 15개 포인트 순회:
  평균 리스폰: 15 x 0.8 = 12개/일
  Common(55%): 6.6개 x 평균 4G = 26G
  Uncommon(30%): 3.6개 x 평균 9G = 32G
  Rare(13%): 1.6개 x 평균 26G = 42G
  Legendary(2%): 0.24개 x 평균 80G = 19G
  일일 기대 수입: ~119G -> 실질(Legendary 조건 미충족 시): ~88G/일

순회 소요 시간: 게임 내 약 30~40분 = 약 1~1.3게임시간
시간당 수익: ~68~88G/게임시간
에너지 소모: 0 (맨손)
```

**수입원 비교 (게임시간 기준)**:

| 활동 | 시간당 수익 | 에너지 소모 | 특성 |
|------|-----------|-----------|------|
| 채집 (Lv.1, 맨손) | ~70G/게임시간 | 0 | 이동만 필요, 에너지 무소모 |
| 채집 (Lv.5, 도구) | ~90G/게임시간 | ~12/순회 | 품질 보정 + 더블 드롭 |
| 낚시 (Lv.1) | ~58G/게임시간 | 2~3/회 | 미니게임 집중 필요 |
| 작물 경작 (옥수수) | ~57G/게임시간 (실작업 기준) | 다수 | 수동 관리 후 자동 성장 |

**포지셔닝 확정 (BAL-016)**: 채집 시간당 수익이 낚시/경작과 유사한 수준으로 조정됨. "에너지 무소모 + 이동만 필요"라는 편의성이 낮은 판매가로 균형을 맞춘다.

### 6.2 일일 수입 상한 비교

| 활동 | 일일 최대 수입 (추정, BAL-016 적용) | 일일 투자 시간 (게임시간) | 비고 |
|------|---------------------|----------------------|------|
| 채집 (전 구역) | ~88~160G | ~1시간 | 포인트 소진 후 더 이상 불가 |
| 낚시 (Lv.1) | ~350~500G (6시간 투자) | ~6시간 | 전념 시 더 높은 총수입 |
| 작물 경작 (20타일) | ~600G/수확일 | ~2시간 (수확일) | 며칠 간격 일괄 수입 |

**포지셔닝 판정**: 채집은 "일일 1시간 이하로 빠르게 순회하여 부수입을 챙기는" 활동이다. 한 번 순회하면 그날은 더 할 것이 없으므로, 나머지 시간을 경작/낚시/가공에 투자하게 된다. 이는 채집이 다른 활동을 **대체하지 않고 보완**하는 구조를 보장한다.

### 6.3 수급 변동

채집물은 `SupplyCategory.Fish`(낚시 생산물과 동일한 보조 수입 카테고리)를 임시 활용하되, 별도 `Forage` 카테고리 추가가 권장된다.

[OPEN] `economy-architecture.md`의 `SupplyCategory` enum (섹션 3.11)에는 현재 `Crop`, `AnimalProduct`, `Fish`, `ProcessedGoods` 4개 값만 존재한다. 채집물 전용 `Forage = 4` 값을 추가하여 독립 수급 파라미터를 적용해야 한다 (FIX-076 범위에 포함 권고).

임시 파라미터 (Fish 카테고리 공유 시):

| 파라미터 | 값 | 비고 |
|----------|-----|------|
| `demandThreshold` | 50 | 주당 50개 이하 판매 시 수급 영향 없음 |
| `supplyDropRate` | 0.005 | 50개 초과 시 매우 느린 가격 하락 |
| `minSupplyMultiplier` | 0.85 | 최대 15% 가격 하락 |

**설계 의도**: 채집물 일일 상한이 ~18개이므로 주간 최대 ~126개. demandThreshold 50을 초과하지만 supplyDropRate가 매우 낮아 실질적으로 가격 하락이 미미하다. 채집은 "꾸준히 팔아도 괜찮은" 안정 수입원.

### 7.2 직접 채집 vs 구매 비교

**강화 낚싯대 재료 (구리 광석 x5)** 확보 방법 비교:

## 10. [RISK] 리스크

1. [RISK] **채집 수입 밸런스**: 채집은 에너지 0(맨손) 또는 1(도구)로 수행 가능하여, 에너지 대비 수익 효율이 낚시/경작보다 월등히 높다. 완화: 일일 채집 가능 수량이 포인트 수(22개소)로 상한이 있어 총수입은 제한된다. 숙련도/품질 보정으로 수입이 증가해도 포인트 상한은 불변.

2. [RISK] **채집 순회의 단조로움**: 매일 동일 경로를 돌며 클릭하는 것이 반복되면 지루해질 수 있다. 완화: 계절마다 아이템이 바뀌고, 날씨에 따라 버섯 리스폰이 달라지며, Legendary 아이템 조건 충족 여부가 매일 달라져 "오늘은 뭐가 있을까" 기대감을 유지한다. 추후 랜덤 이벤트(예: "오늘 숲에서 특이한 냄새가 난다" 힌트) 도입 검토.

3. [RISK] **광석 경쟁으로 인한 자원 병목**: 구리 광석이 채집 낫 강화(x3)와 낚싯대 강화(x5)에 동시에 필요하여, 총 8개를 모아야 한다. 3일 리스폰 + Uncommon 확률이면 약 35~40일 소요. 두 도구를 모두 강화하기까지 기간이 길어 플레이어 답답함 유발 가능. 완화: 여행 상인 구매 경로, 낚시 보물 상자 광석 드롭(-> see `docs/systems/fishing-system.md` 섹션 6.4)으로 병렬 확보 유도.

4. [RISK] **Zone D/E/F 동시 해금 시 콘텐츠 폭주**: Zone D(채집), Zone E(목축), Zone F(낚시)가 레벨 5~6 구간에 몰려 있어 플레이어가 한꺼번에 여러 보조 시스템을 학습해야 할 수 있다. 완화: 해금 순서(Zone D: 레벨 5, Zone F: 레벨 5, Zone E: 레벨 6)에 골드 비용 차이(2,500G vs 3,000G vs 4,000G)를 두어 자연스럽게 시차를 발생시킨다.

5. [RISK] **채집 낫이 도구 슬롯을 추가 점유**: 이미 호미, 물뿌리개, 낫, 낚싯대가 도구 슬롯을 차지하는 상황에서 채집 낫이 추가되면 슬롯 부족 문제. 인벤토리 시스템(-> see `docs/systems/inventory-system.md`)의 도구 슬롯 수가 충분한지 검증 필요.
