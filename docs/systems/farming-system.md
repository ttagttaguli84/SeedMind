# 경작 시스템 (Farming System) 상세 설계

> 작성: Claude Code (Opus) | 2026-04-06  
> 문서 ID: DES-001

---

## Context

이 문서는 SeedMind 핵심 게임 루프의 중심인 경작 시스템을 상세히 기술한다. 마스터 디자인 문서(`docs/design.md`) 섹션 4.1에 정의된 개요를 기반으로, 타일 상태 머신, 도구 인터랙션, 물/비료 메카닉, 토양 품질, 날씨 연동, 도구 업그레이드, 실패 조건, 플레이어 경험까지 모든 세부 사항을 다룬다.

**설계 목표**: 단순한 조작으로 시작하되, 토양 품질/비료/계절 조합을 통해 숙련 플레이어에게 최적화 재미를 제공한다.

---

## 1. 농장 그리드

- **초기 크기**: 8x8 타일 (64타일)
- **최대 크기**: 16x16 타일 (256타일)
- **타일 크기**: 1m x 1m (Unity 월드 단위)
- **확장 방식**: 골드를 지불하여 한 방향으로 4x8 또는 8x4 블록 단위 확장
- **확장 비용**: 1단계 500G, 2단계 1000G, 3단계 2000G, 4단계 4000G (누적 7500G)

## 2. 타일 상태 머신 (Tile State Machine)

### 2.1 상태 목록

| 상태 | 영문 키 | 설명 |
|------|---------|------|
| 빈 땅 | `Empty` | 초기 상태. 잡초/돌이 있을 수 있음 |
| 경작지 | `Tilled` | 호미로 갈아엎은 상태. 씨앗 심기 가능 |
| 심어짐 | `Planted` | 씨앗이 심어진 상태. 즉시 물 필요 |
| 물 줌 | `Watered` | 물을 준 상태. 성장 진행 중 |
| 건조 | `Dry` | 물이 증발한 상태. 성장 정지 |
| 수확 가능 | `Harvestable` | 작물이 최종 단계 도달 |
| 고사 | `Withered` | 작물이 죽은 상태 |
| 건물 | `Building` | 시설물이 배치된 상태 |

### 2.2 상태 전환 다이어그램

```
                    ┌──────────┐
                    │  Empty   │
                    └────┬─────┘
                         │ [호미 사용]
                         ▼
                    ┌──────────┐   3일 방치(씨앗 없이)
                    │  Tilled  │ ──────────────────────┐
                    └────┬─────┘                       │
                         │ [씨앗 사용]                  ▼
                         ▼                        ┌──────────┐
                    ┌──────────┐                  │  Empty   │
                    │ Planted  │                  └──────────┘
                    └────┬─────┘
                         │ [물뿌리개 사용 / 비]
                         ▼
                    ┌──────────┐  물 증발 (다음날 아침)
                    │ Watered  │ ─────────────────┐
                    └────┬─────┘                  │
                         │ 성장 완료               ▼
                         │                   ┌──────────┐
                         │                   │   Dry    │
                         │                   └────┬─────┘
                         │                        │ [물뿌리개 사용 / 비]
                         │                        │ → Watered로 복귀
                         │                        │
                         │                        │ 3일 연속 Dry
                         │                        ▼
                         │                   ┌──────────┐
                         │                   │ Withered │
                         │                   └────┬─────┘
                         │                        │ [낫 사용]
                         │                        ▼
                         │                   ┌──────────┐
                         │                   │  Tilled  │
                         ▼                   └──────────┘
                    ┌──────────────┐
                    │ Harvestable  │
                    └──────┬───────┘
                           │ [낫 사용 / 손 상호작용]
                           ▼
                      ┌──────────┐
                      │  Tilled  │  (수확 후 경작지 유지)
                      └──────────┘
```

### 2.3 상태 전환 규칙 상세

| 전환 | 조건 | 결과 |
|------|------|------|
| Empty → Tilled | 호미 사용 | 타일 비주얼 변경, 잡초/돌 제거 |
| Tilled → Empty | 씨앗 없이 3일 경과 | 자연 복구 (비주얼: 풀이 다시 자람) |
| Tilled → Planted | 씨앗 아이템 사용 | 씨앗 인벤토리 -1, 작물 데이터 바인딩 |
| Planted → Watered | 물뿌리개 사용 또는 비 | 성장 타이머 시작 |
| Watered → Dry | 다음 날 06:00 (새벽) | 물 증발, 성장 타이머 일시정지 |
| Dry → Watered | 물뿌리개 사용 또는 비 | 성장 타이머 재개 |
| Dry → Withered | 연속 3일 Dry 상태 유지 | 작물 사망, 수확 불가 |
| Watered → Harvestable | 성장일수 충족 | 작물 최종 비주얼, 수확 가능 |
| Harvestable → Tilled | 낫 사용 또는 E키 | 작물 아이템 인벤토리에 추가 |
| Withered → Tilled | 낫 사용 | 고사체 제거, 아이템 없음 |
| 모든 상태 → Building | 건설 모드에서 시설 배치 | 기존 작물 파괴 (확인 팝업) |

## 3. 도구 인터랙션 (Tool Interactions)

### 3.1 도구 목록

| 도구 | 슬롯 | 기본 범위 | 에너지 소모 | 사용 대상 |
|------|-------|-----------|-------------|-----------|
| 호미 (Hoe) | 1 | 1타일 | 2 | Empty 타일 |
| 물뿌리개 (Watering Can) | 2 | 1타일 | 1 | Planted, Dry 타일 |
| 씨앗봉투 (Seed Bag) | 3 | 1타일 | 0 | Tilled 타일 |
| 낫 (Sickle) | 4 | 1타일 | 1 | Harvestable, Withered 타일 |
| 손 (Hand) | 5 | - | 0 | 상호작용 전용 (상점, 시설, 아이템 줍기) |
| 비료 (Fertilizer) | 별도 | 1타일 | 0 | Tilled, Planted, Dry 타일 |

### 3.2 에너지 시스템

- **최대 에너지**: 100 (기본)
- **회복**: 수면 시 전량 회복, 음식 섭취로 부분 회복
- **에너지 0**: 도구 사용 불가, 이동 속도 50% 감소
- **에너지 경고**: 20 이하일 때 UI 경고 표시

### 3.3 도구별 상세

#### 호미 (Hoe)
- **사용 조건**: Empty 타일 위에서 좌클릭
- **효과**: 타일을 Tilled 상태로 전환
- **애니메이션**: 캐릭터가 땅을 파는 모션 (0.5초)
- **쿨다운**: 0.6초 (애니메이션 포함)
- **잘못된 대상**: Tilled 이상 타일에 사용 시 "이미 경작된 땅입니다" 피드백

#### 물뿌리개 (Watering Can)
- **사용 조건**: Planted 또는 Dry 상태 타일
- **효과**: 타일을 Watered 상태로 전환, 해당 날 성장 진행
- **용량**: 기본 20회분, 물가/우물에서 리필
- **리필 시간**: 1초 (물가 인접 시 자동 표시)
- **애니메이션**: 물 뿌리는 모션 (0.4초)
- **쿨다운**: 0.5초
- **빈 물뿌리개 사용 시**: "물을 채워야 합니다" 피드백, 가까운 수원 방향 표시

#### 씨앗봉투 (Seed Bag)
- **사용 조건**: Tilled 타일 + 인벤토리에 해당 씨앗 보유
- **효과**: 타일을 Planted 상태로 전환, 작물 종류 바인딩
- **씨앗 선택**: 씨앗봉투 선택 후 마우스 휠로 씨앗 종류 변경 (보유 씨앗만 표시)
- **계절 확인**: 현재 계절에 심을 수 없는 씨앗은 경고 표시 (심기 자체는 가능하지만 성장 불가)
- **애니메이션**: 씨앗 뿌리는 모션 (0.3초)
- **쿨다운**: 0.4초

#### 낫 (Sickle)
- **사용 조건**: Harvestable 또는 Withered 타일
- **효과**:
  - Harvestable: 작물 수확, 인벤토리에 아이템 추가, 경험치 획득
  - Withered: 고사체 제거, 아이템/경험치 없음
- **수확량**: 작물별 기본 수확량 1개 (토양 품질/비료에 따라 변동, 섹션 5 참조)
- **애니메이션**: 베는 모션 (0.4초)
- **쿨다운**: 0.5초

### 3.4 잘못된 도구 사용 피드백

플레이어가 부적절한 타일에 도구를 사용할 경우:
- 캐릭터 머리 위에 "?" 또는 "X" 아이콘 (0.5초)
- 짧은 효과음 (부정적 톤, "뿅" 계열)
- 도구 액션 실행하지 않음 (에너지 소모 없음)

## 4. 물 메카닉 (Water Mechanics)

### 4.1 기본 규칙

- **물주기 시점**: Planted 또는 Dry 상태에서 물을 줄 수 있음
- **물 지속시간**: 물을 준 날의 다음 날 06:00에 증발
- **하루 1회 원칙**: 같은 날 물을 여러 번 줘도 추가 효과 없음 (에너지만 소모)
- **성장 조건**: 작물은 Watered 상태인 날만 성장일수가 1일 카운트됨

### 4.2 증발 타이밍

| 시점 | 이벤트 |
|------|--------|
| 아무 때나 | 플레이어가 물을 줌 → Watered |
| 다음 날 06:00 | 물 증발 → Dry (비가 오지 않는 한) |
| 06:00~24:00 | 다시 물을 줘야 성장 진행 |

### 4.3 물뿌리개 용량

등급별 용량 및 범위 (→ see `docs/systems/tool-upgrade.md` 섹션 3.2 for canonical 수치):

| 등급 | 용량 (회) | 범위 |
|------|-----------|------|
| Basic | 20 | 1x1 |
| Reinforced | 40 | 1x3 |
| Legendary | 80 | 3x3 |

## 5. 비료 메카닉 (Fertilizer Mechanics)

### 5.1 비료 종류

| 비료 | 구매가 | 효과 | 지속 | 해금 |
|------|--------|------|------|------|
| 기본 비료 (Basic Fertilizer) | 20G | 성장 속도 +25% | 수확까지 | 시작 |
| 고급 비료 (Quality Fertilizer) | 50G | 고품질 수확 확률 +30% | 수확까지 | 레벨 3 |
| 속성장 비료 (Speed-Gro) | 40G | 성장 속도 +50% | 수확까지 | 레벨 4 |
| 유기 비료 (Organic Fertilizer) | 80G | 토양 품질 +1단계 (영구) | 영구 | 레벨 6 |

### 5.2 사용 규칙

- **적용 시점**: Tilled, Planted, Dry 상태에서 적용 가능 (Watered도 가능)
- **중첩 규칙**: 한 타일에 **하나의 비료만** 적용 가능. 새 비료 적용 시 기존 비료 대체
- **지속**: 수확 시 비료 효과 소멸 (유기 비료 제외)
- **비주얼**: 비료가 적용된 타일은 색상 변화로 표시 (기본=밝은 갈색, 고급=보라 입자, 속성장=초록 입자, 유기=짙은 갈색)

### 5.3 성장 속도 계산

```
실제_성장일수 = ceil(기본_성장일수 / (1.0 + 비료_속도보너스 + 토양_보너스))
```

**예시**: 토마토 (기본 5일) + 속성장 비료 (+50%)
- 실제 성장일수 = ceil(5 / 1.5) = ceil(3.33) = 4일

### 5.4 수확 품질 시스템

| 품질 | 가격 배수 | 비주얼 |
|------|-----------|--------|
| 일반 (Normal) | x1.0 | 별 없음 |
| 은별 (Silver) | x1.25 | 은색 별 1개 |
| 금별 (Gold) | x1.5 | 금색 별 1개 |
| 이리듐 (Iridium) | x2.0 | 보라색 별 1개 |

**품질 결정 공식**:

```
품질_점수 = 토양_품질_보너스 + 비료_품질_보너스 + random(0, 0.3)

일반:    품질_점수 < 0.3
은별:    0.3 <= 품질_점수 < 0.6
금별:    0.6 <= 품질_점수 < 0.9
이리듐:  품질_점수 >= 0.9
```

## 6. 토양 품질 시스템 (Soil Quality)

### 6.1 품질 등급

| 등급 | 영문 키 | 성장 속도 보너스 | 품질 보너스 | 비주얼 |
|------|---------|-----------------|-------------|--------|
| 척박 | `Poor` | -10% | +0.0 | 밝은 회갈색 |
| 보통 | `Normal` | 0% | +0.1 | 갈색 |
| 비옥 | `Fertile` | +10% | +0.25 | 짙은 갈색 |
| 풍요 | `Rich` | +20% | +0.4 | 거의 검은 갈색 |

### 6.2 토양 품질 변화

| 행위 | 효과 |
|------|------|
| 같은 작물 연속 재배 (3회+) | 품질 -1단계 (연작 피해) |
| 작물 종류 교체 | 품질 유지 |
| 한 계절 휴경 (빈 Tilled 상태 유지) | 품질 +1단계 (최대 Normal) |
| 유기 비료 사용 | 품질 +1단계 (최대 Rich) |
| 계절 전환 시 작물 미수확 방치 | 품질 -1단계 |

### 6.3 연작 피해 (Crop Rotation)

- **추적**: 타일별로 최근 재배 작물 종류와 연속 횟수를 기록
- **트리거**: 같은 작물을 3회 연속 심으면 토양 품질 1단계 하락
- **해제**: 다른 작물을 1회 재배하면 연속 카운터 리셋
- **플레이어 정보**: 타일 정보 팝업에 "연속 재배: 감자 x2" 형태로 표시
- **의미**: 플레이어에게 작물 다양화 동기 부여 (지배 전략 방지)

[OPEN] 연작 피해가 초보 플레이어에게 너무 가혹할 수 있음. 튜토리얼 단계에서 면제하거나, 최초 경고를 주는 방식 검토 필요.

## 7. 도구 업그레이드 (Tool Upgrades)

도구 업그레이드 시스템의 상세 설계는 독립 문서로 분리되었다.

**Canonical 문서**: `docs/systems/tool-upgrade.md` (DES-007)

해당 문서에서 다루는 내용:
- 3단계 등급 체계 (Basic / Reinforced / Legendary)
- 도구별 업그레이드 성능 수치 (범위, 에너지, 특수 효과)
- 업그레이드 비용 (골드 + 재료), 레벨 요건, 소요 시간
- 대장간 NPC 상호작용 흐름
- 에너지 효율 비교, ROI 분석, 전략적 트레이드오프
- 시각적 변화, UI 표시

> **참고**: 기존 5단계(Basic/Copper/Iron/Gold/Iridium) 체계는 3단계로 재설계됨. 광산 시스템 없이 대장간 상점에서 재료를 구매하는 방식으로 변경. 상세 근거는 tool-upgrade.md Context 섹션 참조.

## 8. 날씨 연동 (Weather Integration)

### 8.1 날씨 종류

(→ see `docs/systems/time-season.md` 섹션 3.1~3.2 for canonical 날씨 종류 및 계절별 확률 테이블, DES-003)

날씨는 7종(`Clear`, `Cloudy`, `Rain`, `HeavyRain`, `Storm`, `Snow`, `Blizzard`)으로 정의되며, 계절별 확률과 영문 키는 `docs/systems/time-season.md` 섹션 3.1이 canonical source이다.

### 8.2 날씨의 경작 영향

| 날씨 (영문 키) | 효과 |
|--------------|------|
| `Clear` | 효과 없음 (기본 상태) |
| `Cloudy` | 효과 없음 |
| `Rain` | **모든 경작 타일 자동 물주기**, 물뿌리개 사용 불필요 |
| `HeavyRain` | 자동 물주기 |
| `Storm` | 자동 물주기 + **10% 확률로 작물 피해** (성장 단계 -1, → see `docs/systems/time-season.md` 섹션 5.3 `stormDamageChance`) |
| `Snow` | 야외 작물 성장 정지 (온실 내부 제외) |
| `Blizzard` | 야외 작물 성장 정지 + 5% 확률 작물 동사(Withered) |

(→ see `docs/systems/time-season.md` 섹션 3.4 for 전체 날씨 영향 상세)

### 8.3 날씨 예보

- 다음 날 날씨는 전날 저녁(18:00)에 TV/게시판에서 확인 가능
- 날씨는 하루 단위로 결정 (06:00 전환)
- 비 오는 날은 에너지 절약의 기회 → 다른 활동(탐험, 건설 등)에 시간 활용

## 9. 실패 조건 (Failure Conditions)

### 9.1 작물 고사

| 조건 | 결과 | 복구 |
|------|------|------|
| 3일 연속 물 미제공 (Dry 유지) | Withered 상태 전환 | 낫으로 제거 → Tilled |
| 계절 불일치 (계절 전환 시 미수확) | 즉시 Withered | 낫으로 제거 → Tilled |
| 폭풍 피해 | 10% 확률 성장 단계 -1 | 계속 키우면 회복 (추가 일수 필요) |

### 9.2 계절 전환 규칙

- 계절이 바뀌면, 해당 계절에 맞지 않는 **모든 야외 작물**이 즉시 Withered
- 플레이어에게 계절 전환 **3일 전** 경고 표시: "3일 후 여름이 됩니다. 봄 작물을 수확하세요!"
- 온실 내부 작물은 계절 전환 영향 없음 (→ see `docs/design.md` 섹션 4.6, 온실)

### 9.3 작물별 계절 매핑

| 작물 | 재배 가능 계절 |
|------|---------------|
| 감자 | 봄 |
| 당근 | 봄, 가을 |
| 토마토 | 여름 |
| 옥수수 | 여름 |
| 딸기 | 봄 |
| 호박 | 가을 |
| 해바라기 | 여름 |
| 수박 | 여름 |

(→ see `docs/content/crops.md` 작성 예정, CON-001에서 전체 작물 목록 확정)

[OPEN] 다중 계절 작물(예: 당근 봄+가을)의 계절 전환 처리 방식. 봄에 심은 당근이 여름에 고사하는지, 가을까지 생존하는지 결정 필요. 현재 설계: 여름에 고사 (계절 불일치 규칙 적용).

## 10. 플레이어 경험 (Player Experience)

### 10.1 행동별 피드백

| 행동 | 시각 피드백 | 청각 피드백 | 촉각(진동) |
|------|------------|------------|-----------|
| 호미질 | 흙 파티클, 타일 색상 변화 | "쓱" 파는 소리 | - |
| 씨앗 심기 | 씨앗 떨어지는 애니메이션 | 가벼운 "톡" | - |
| 물주기 | 물방울 파티클, 타일 색상 어두워짐 | "철철" 물소리 | - |
| 비료 주기 | 입자 스프레이 효과 | 가루 뿌리는 "솨" | - |
| 수확 | 작물 팝 효과 + 아이템 아이콘 날아감 | "뽑" + 경쾌한 징글 | - |
| 고사 발견 | 시든 비주얼 (갈변) | - | - |
| 레벨업 | 전체 화면 빛 효과 | 팡파레 | - |

### 10.2 정보 표시

- **타일 호버**: 마우스를 타일 위에 올리면 미니 팝업 표시
  - 작물 이름, 현재 성장 단계 (예: "토마토 - 새싹 (2/5일)")
  - 물 상태 (물방울 아이콘: 파랑=물 줌, 빨강=건조)
  - 비료 종류 (아이콘)
  - 토양 품질 (별 표시: Poor=0, Normal=1, Fertile=2, Rich=3)
  - 연속 재배 경고 (해당 시)

### 10.3 만족감 설계 포인트

1. **즉각적 반응**: 모든 도구 사용에 0.1초 이내 시각/청각 피드백
2. **진행 가시성**: 작물 성장이 단계별로 뚜렷하게 보임 (4단계 각각 모델/스프라이트 변화)
3. **수확의 쾌감**: 수확 시 "팝" 효과 + 골드/경험치 표시 + 연속 수확 콤보 사운드
4. **계획의 보상**: 비료+토양 관리를 잘한 플레이어는 금별/이리듐 품질 수확 → 높은 수익
5. **실수의 관대함**: Withered 작물은 낫으로 즉시 정리 가능, 토양은 보존됨

### 10.4 연속 수확 보너스

- 5개 이상 연속 수확 시 콤보 표시 ("x5 수확!")
- 콤보 보너스: 경험치 +10% (수확물 가격에는 영향 없음)
- 콤보 리셋: 2초간 수확하지 않으면 리셋

## Risks

1. [RISK] **시스템 복잡도 과부하**: 토양 품질 + 비료 + 날씨 + 계절 + 연작이 동시에 작용하면 초보 플레이어가 압도당할 수 있음. 단계적 해금으로 완화 필요 (비료는 레벨 2+, 토양 표시는 레벨 3+).
2. [RISK] **물주기 반복 노동**: 매일 전체 농장에 물주기가 지루해질 수 있음. 물탱크/스프링클러 해금 시점이 중요.
3. [RISK] **도구 업그레이드 재료**: 광산 미구현 시 업그레이드 경로가 단조로워질 수 있음.
