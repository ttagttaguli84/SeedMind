# 도구 업그레이드 시스템 (Tool Upgrade System) 상세 설계

> 작성: Claude Code (Opus) | 2026-04-06  
> 문서 ID: DES-007

---

## Context

이 문서는 SeedMind의 도구 업그레이드 시스템을 상세히 기술한다. 플레이어가 호미, 물뿌리개, 낫 3종의 도구를 단계적으로 강화하여 경작 효율을 높여가는 성장 경험을 설계한다.

**이 문서는 도구 업그레이드 수치의 canonical 문서이다.** `docs/systems/farming-system.md` 섹션 7은 본 문서를 참조하며, 기존 5단계(Basic/Copper/Iron/Gold/Iridium) 체계를 3단계(기본/강화/전설)로 재설계한다.

**재설계 근거**:
- 기존 5단계는 광산 시스템(광석 채굴)을 전제로 했으나, 광산은 현재 스코프 밖이다 (-> see `docs/systems/farming-system.md` Open Question 1)
- 3단계로 축소하면 광석 재료 없이 골드 + 간단한 재료만으로 업그레이드 가능
- 플레이어 경험 관점에서 3단계가 "기본 -> 중반 -> 후반" 진행 리듬에 자연스럽게 대응

**설계 목표**: 도구 업그레이드가 농장 관리의 "반복 노동 경감"과 "효율 극대화" 사이에서 의미 있는 투자 결정이 되도록 한다. 어떤 도구를 먼저 올릴지에 따라 플레이 스타일이 달라진다.

## 1. 업그레이드 등급 체계

### 1.1 등급 정의

| 등급 | 영문 키 | 레벨 구간 | 설계 의도 |
|------|---------|-----------|-----------|
| 기본 (Lv.1) | `Basic` | 레벨 1~3 | 게임 시작 시 지급. 한 타일씩 작업하며 기본 메카닉 학습 |
| 강화 (Lv.2) | `Reinforced` | 레벨 3~6 | 농장 확장기. 범위 확대와 용량 증가로 반복 노동 경감 |
| 전설 (Lv.3) | `Legendary` | 레벨 7~10 | 후반 최적화. 넓은 범위 + 특수 효과로 대규모 농장 운영 |

### 1.2 등급과 기존 시스템 매핑

`docs/systems/inventory-system.md`의 `toolTier` 속성을 다음과 같이 재정의한다:

| 기존 값 | 신규 값 | 비고 |
|---------|---------|------|
| `Basic` | `Basic` | 변경 없음 |
| `Copper` | (삭제) | 강화로 통합 |
| `Iron` | `Reinforced` | 등급명 변경 |
| `Gold` | `Legendary` | 등급명 변경 |
| `Iridium` | (삭제) | 스코프 축소로 제거 |

## 2. 업그레이드 조건 및 비용

### 2.1 공통 업그레이드 비용

| 업그레이드 | 골드 비용 | 필요 재료 | 플레이어 레벨 요건 | 소요 시간 |
|-----------|----------|-----------|------------------|-----------|
| Basic -> Reinforced | 800G | 철 조각(Iron Scrap) x3 | 레벨 3 | 1일 |
| Reinforced -> Legendary | 3,000G | 정제 강철(Refined Steel) x3 | 레벨 7 | 2일 |

**비용 근거**:
- Basic -> Reinforced (800G): 봄 시즌(28일) 수입 시뮬레이션에서 일반 플레이어의 누적 수입이 약 2,000~3,000G (-> see `docs/balance/progression-curve.md` 섹션 3). 레벨 3 도달 시점에 도구 1개 업그레이드가 가능한 수준. 3종 모두 올리려면 2,400G로 시설 투자(물탱크 500G, 창고 1,000G)와 트레이드오프 발생
- Reinforced -> Legendary (3,000G): 레벨 7 도달 시점(여름~가을)에 충분한 자금이 축적된 상태. 가공소(3,000G)와 동일한 가격대로, 가공 투자 vs 도구 투자 선택 유도

### 2.2 재료 획득 경로

광산 시스템 없이도 재료를 획득할 수 있도록 설계한다.

| 재료 | 영문 키 | 획득 방법 | 비고 |
|------|---------|-----------|------|
| 철 조각 | `iron_scrap` | 대장간 상점에서 구매 (100G/개), 또는 **채집 철 광석 x3을 가공소에서 제련** (DES-020) | 상점: 3개 = 300G. 제련: 3개 = 철 광석 9개(직판 포기 108G) + 가공 6시간 |
| 정제 강철 | `refined_steel` | 대장간 상점에서 구매 (400G/개) | 3개 = 1,200G, 업그레이드 총비용 4,200G |

**철 조각 제련 경로 (DES-020 확정)**: 채집으로 얻은 철 광석(`gather_iron_ore`) x3을 가공소(일반)에서 제련하여 철 조각 x1을 생산할 수 있다. 상점 구매(100G/개, 즉시)와 제련(기회비용 36G + 가공 2시간, 채집 ~25일)의 트레이드오프. 레시피 상세: (-> see `docs/content/processing-system.md` 섹션 3.7.4).

### 2.3 업그레이드 절차

1. 플레이어가 대장간(Blacksmith) NPC에게 말을 건다
2. 업그레이드 메뉴에서 도구와 등급을 선택한다
3. 골드 + 재료가 충분하고 레벨 요건을 충족하면 의뢰 가능
4. 의뢰 기간 동안 해당 도구는 **사용 불가** (대장간에 맡긴 상태)
5. 소요 시간 경과 후 대장간을 방문하여 완성된 도구를 수령한다

**도구 부재 기간의 설계 의도**: 업그레이드 타이밍을 전략적으로 고르게 만든다. 비 오는 날(물뿌리개 불필요)에 물뿌리개를 맡기거나, 수확 직후(낫 불필요 기간)에 낫을 맡기는 등 플레이어의 계획 능력을 보상한다.

### 2.4 채집 낫 업그레이드 조건 (GatheringMastery 기반) (FIX-086)

채집 낫은 기존 농업 도구(호미/물뿌리개/낫)와 달리 **채집 숙련도(GatheringMastery)** 를 해금 조건으로 사용한다. 플레이어 메인 레벨이 아닌 채집 숙련도 레벨을 요구하므로, 업그레이드 스키마에 `levelReqType` 필드가 추가되었다 (→ see `docs/systems/tool-upgrade-architecture.md` 섹션 3.6).

**해금 조건 타입 비교**:

| 도구 종류 | levelReqType | requiredLevel 의미 | 비고 |
|-----------|-------------|-------------------|------|
| 호미, 물뿌리개, 낫 (농업) | `PlayerLevel` (기본값) | 플레이어 메인 레벨 | 섹션 2.1 참조 |
| 채집 낫 | `GatheringMastery` | 채집 숙련도 레벨 | → see `docs/systems/gathering-system.md` 섹션 4, 5.2 |

**채집 낫 등급별 해금 조건 요약**:

채집 낫의 등급, 비용, 재료, 해금 조건 상세는 `docs/systems/gathering-system.md` 섹션 5.2가 canonical이다. 여기서는 해금 조건 타입의 차이만 명시한다.

| 업그레이드 | levelReqType | requiredLevel | 비고 |
|-----------|-------------|:-------------:|------|
| 없음 -> Basic | (구매, 업그레이드 아님) | Zone D 해금 | → see `docs/systems/gathering-system.md` 섹션 5.2 |
| Basic -> Reinforced | `GatheringMastery` | (→ see `docs/systems/gathering-system.md` 섹션 5.2) | 채집 숙련도 기반 |
| Reinforced -> Legendary | `GatheringMastery` | (→ see `docs/systems/gathering-system.md` 섹션 5.2) | 채집 숙련도 기반 |

**UI 표시 차이**: 대장간 업그레이드 메뉴에서 채집 낫의 해금 조건은 "플레이어 레벨 N" 대신 "채집 숙련도 Lv.N"으로 표시해야 한다. `levelReqType`에 따라 UI 텍스트를 분기한다 (→ see `docs/systems/tool-upgrade-architecture.md` 섹션 5.1).

**도구 부재 기간 차이**: 채집 낫 업그레이드 중에도 맨손 채집이 가능하므로, 농업 도구에 비해 도구 부재 페널티가 완화된다 (→ see `docs/systems/gathering-system.md` 섹션 5.3).

## 3. 도구별 업그레이드 상세

### 3.1 호미 (Hoe)

경작지를 만드는 도구. 업그레이드 시 범위 확대와 장애물 자동 제거로 농장 확장이 수월해진다.

| 속성 | Basic (Lv.1) | Reinforced (Lv.2) | Legendary (Lv.3) |
|------|-------------|-------------------|------------------|
| 범위 | 1x1 (1타일) | 1x3 (가로줄) | 3x3 (9타일) |
| 에너지 소모 | 2 | 3 | 5 |
| 쿨다운 | 0.6초 | 0.8초 | 1.2초 |
| 특수 효과 | - | 돌 자동 제거 | 돌+잡초 자동 제거 |
| 애니메이션 | 땅 파기 (단일) | 넓게 긁기 (가로) | 내려찍기 (충격파) |

**밸런스 분석**:
- Basic: 1타일당 에너지 2. 에너지 100 기준 50타일 경작 가능
- Reinforced: 3타일당 에너지 3, 즉 1타일당 에너지 1. 효율 2배
- Legendary: 9타일당 에너지 5, 즉 1타일당 에너지 0.56. 효율 약 3.6배

### 3.2 물뿌리개 (Watering Can)

작물에 물을 주는 도구. 업그레이드 시 저수량 증가와 범위 확대로 일일 물주기 부담이 크게 줄어든다.

| 속성 | Basic (Lv.1) | Reinforced (Lv.2) | Legendary (Lv.3) |
|------|-------------|-------------------|------------------|
| 범위 | 1x1 (1타일) | 1x3 (가로줄) | 3x3 (9타일) |
| 저수량 (회) | 20 | 40 | 80 |
| 에너지 소모 | 1 | 2 | 3 |
| 쿨다운 | 0.5초 | 0.6초 | 0.9초 |
| 리필 시간 | 1.0초 | 1.5초 | 2.0초 |
| 특수 효과 | - | - | 물 준 타일에 성장 속도 +5% 미량 보너스 |
| 애니메이션 | 물 뿌리기 (단일) | 물 뿌리기 (가로) | 물 쏟기 (넓은 범위) |

**밸런스 분석**:
- Basic: 20타일 물주기 가능 (에너지 20 소모). 리필 1회로 초기 농장(8x8중 활용 20타일) 커버
- Reinforced: 40타일분 저수, 3타일씩 물주기. 에너지 약 27 소모로 40타일 커버 (ceil(40/3)*2)
- Legendary: 80타일분 저수, 9타일씩 물주기. 에너지 약 27 소모로 80타일 커버. 최대 농장(256타일)은 리필 3~4회 필요

**물탱크 시설과의 역할 분담**: 물탱크(-> see `docs/design.md` 섹션 4.6, 500G, 레벨 3)는 인접 타일만 자동 물주기. 물뿌리개 업그레이드는 플레이어가 직접 물주는 범위/효율 강화. 두 시스템은 "자동화 범위 vs 수동 효율"로 보완 관계. 물탱크 범위 밖 타일은 물뿌리개로 커버해야 하므로, 양쪽 모두 투자 가치가 있다.

### 3.3 낫 (Sickle)

작물 수확 및 고사체 제거 도구. 업그레이드 시 범위 확대와 보너스 수확 확률로 수익을 직접 높인다.

| 속성 | Basic (Lv.1) | Reinforced (Lv.2) | Legendary (Lv.3) |
|------|-------------|-------------------|------------------|
| 범위 | 1x1 (1타일) | 1x3 (가로줄) | 3x3 (9타일) |
| 에너지 소모 | 1 | 1 | 2 |
| 쿨다운 | 0.5초 | 0.6초 | 0.9초 |
| 보너스 수확 확률 | 0% | 15% (추가 +1개) | 25% (추가 +1개) |
| 품질 상승 확률 | 0% | 0% | 10% (품질 +1등급) |
| 특수 효과 | - | - | 수확 시 씨앗 회수 확률 10% |
| 애니메이션 | 베기 (단일) | 가로 베기 | 회전 베기 (넓은 범위) |

**밸런스 분석**:
- Basic: 단순 수확. 1타일 1에너지
- Reinforced: 3타일 1에너지. 효율 3배. 15% 보너스 수확은 기대값으로 수확량 약 1.15배
- Legendary: 9타일 2에너지. 효율 약 4.5배. 25% 보너스 수확 + 10% 품질 상승 + 10% 씨앗 회수. 기대값으로 수확 가치 약 1.35~1.45배 (작물/품질에 따라 변동)

**낫의 보너스 수확과 작물 수확량의 관계**: 기본 수확량은 1개이다 (-> see `docs/systems/farming-system.md` 섹션 3.3). 보너스 수확은 확률적으로 추가 1개를 드롭하는 것이며, 기본 수확량 자체를 변경하지 않는다. 토양 품질/비료에 의한 수확량 보너스와는 독립적으로 판정한다.

## 4. 에너지 효율 비교표

도구 업그레이드별 타일당 에너지 효율을 종합 비교한다. 에너지 소모 기본값은 (-> see `docs/systems/farming-system.md` 섹션 3.2, 최대 에너지 100).

| 도구 | 등급 | 범위 (타일) | 에너지/1회 | 타일당 에너지 | 효율 배수 (vs Basic) |
|------|------|------------|-----------|-------------|---------------------|
| 호미 | Basic | 1 | 2 | 2.00 | x1.0 |
| 호미 | Reinforced | 3 | 3 | 1.00 | x2.0 |
| 호미 | Legendary | 9 | 5 | 0.56 | x3.6 |
| 물뿌리개 | Basic | 1 | 1 | 1.00 | x1.0 |
| 물뿌리개 | Reinforced | 3 | 2 | 0.67 | x1.5 |
| 물뿌리개 | Legendary | 9 | 3 | 0.33 | x3.0 |
| 낫 | Basic | 1 | 1 | 1.00 | x1.0 |
| 낫 | Reinforced | 3 | 1 | 0.33 | x3.0 |
| 낫 | Legendary | 9 | 2 | 0.22 | x4.5 |

**설계 의도**: 낫은 수확(보상 행위)이므로 효율 개선폭이 가장 크다. 호미는 농장 확장 시에만 대량 사용하므로 중간 수준. 물뿌리개는 매일 사용하므로 총 절약 에너지가 가장 크지만, 배율 자체는 적당히 억제한다.

## 5. 업그레이드 우선순위 분석 (전략적 트레이드오프)

### 5.1 플레이어별 추천 전략

| 전략 | 첫 업그레이드 | 근거 |
|------|-------------|------|
| 효율 극대화 | 물뿌리개 | 매일 사용하므로 에너지 절감 효과가 누적됨 |
| 수익 극대화 | 낫 | 보너스 수확으로 판매 수입 직접 증가 |
| 확장 우선 | 호미 | 새 경작지를 빠르게 개간하여 농장 규모 조기 확대 |

### 5.2 비용 대비 회수 기간 (ROI)

Reinforced 업그레이드 기준 (총 비용 1,100G = 골드 800G + 재료 300G):

| 도구 | 일일 절감 가치 | 회수 기간 | 산출 근거 |
|------|--------------|-----------|-----------|
| 물뿌리개 | 에너지 약 7 절감 (20타일 기준) | 간접적 (에너지 → 추가 경작 → 추가 수입) | 절감 에너지로 추가 3~4타일 운영 가능, 일 50~80G 추가 수입 추정. 약 15~22일 회수 |
| 낫 | 수확당 +15% 확률 추가 드롭 | 약 25~35일 | 일 평균 10타일 수확, 기대 추가 1.5개, 평균 작물가 40G 기준 일 60G. 약 18일 회수 |
| 호미 | 확장 시에만 효과 | 측정 불가 (확장 시점 의존) | 농장 확장을 계획하는 경우에만 투자 가치 |

**결론**: 물뿌리개와 낫의 ROI가 비슷하므로 플레이 스타일에 따른 선택이 유의미하다. 호미는 확장기에만 가치가 있어 자연스럽게 후순위가 되지만, 농장을 빠르게 키우려는 플레이어에게는 1순위가 될 수 있다. **지배적 전략 없음** 확인.

## 7. 시각적 변화

### 7.1 도구 외형 변화

| 도구 | Basic | Reinforced | Legendary |
|------|-------|-----------|-----------|
| 호미 | 나무 손잡이 + 갈색 금속 날 | 나무 손잡이 + 짙은 회색 강철 날, 날이 더 넓음 | 장식된 검은 손잡이 + 은빛 큰 날, 미세한 발광 이펙트 |
| 물뿌리개 | 작은 갈색 양철통 | 중간 크기 회색 강철통, 분사구 확대 | 큰 은빛 통, 장식 문양, 물줄기에 미세 파티클 |
| 낫 | 짧은 나무 손잡이 + 작은 곡선 날 | 긴 손잡이 + 넓은 강철 곡선 날 | 장식 손잡이 + 크고 날카로운 은빛 날, 베기 시 잔상 이펙트 |

### 7.2 사용 이펙트 변화

| 등급 | 이펙트 |
|------|--------|
| Basic | 기본 파티클 (흙 튀김, 물방울, 풀 파편) |
| Reinforced | 기본 + 범위 표시 가이드라인 (반투명 사각형) + 약간 더 화려한 파티클 |
| Legendary | 기본 + 범위 표시 + 충격파/빛 이펙트 + 사운드 강화 |

### 7.3 UI 표시

도구바 슬롯에서 등급을 시각적으로 구분한다 (-> see `docs/systems/inventory-system.md` 섹션 4.3 도구 등급 표시):

| 등급 | 슬롯 테두리 색상 | 등급 아이콘 |
|------|----------------|------------|
| Basic | 없음 (기본) | 없음 |
| Reinforced | 파란색 테두리 | 작은 파란 화살표 |
| Legendary | 금색 테두리 + 미세 발광 | 작은 금색 별 |

## Risks

1. [RISK] **도구 부재 기간 불만**: 업그레이드 중 도구를 사용할 수 없어 초보 플레이어가 답답해할 수 있음. 완화책: 업그레이드 전 명확한 경고 UI + 전략 팁("비 오는 날에 물뿌리개를 맡기면 좋아요").
2. [RISK] **범위 도구의 밸런스 불안정**: 3x3 범위가 강력하여 Legendary 도구 획득 이후 게임이 너무 쉬워질 수 있음. 에너지 소모 증가와 쿨다운 증가로 완화했으나, 실제 플레이에서 검증 필요.
3. [RISK] **광산 시스템 부재로 인한 재료 획득의 단조로움**: 상점 구매만으로 재료를 얻으면 업그레이드 과정이 "골드만 모으면 됨"으로 단순화됨. 재료 드롭 경로 추가(Open Question 1)로 다양성 확보 필요.
4. [RISK] **기존 문서와의 불일치**: farming-system.md 섹션 7의 5단계 체계, inventory-system.md의 toolTier 값(Basic/Copper/Iron/Gold), progression-curve.md의 도구 XP 계산이 모두 수정 필요. 본 문서 확정 후 일괄 업데이트 대상.
