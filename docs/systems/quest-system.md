## Context

이 문서는 SeedMind의 퀘스트/미션 시스템을 상세히 기술한다. 플레이어에게 단기/중기/장기 목표를 제시하여, 자유도가 높은 농장 시뮬레이션에서 "다음에 무엇을 할지" 방향성을 제공하면서도, 목표 달성 방식의 선택권을 플레이어에게 보장한다.

**설계 목표**:
- 핵심 게임 루프(심기 -> 관리 -> 수확 -> 판매 -> 확장)에 "목적"과 "보상"을 부여
- 계절/레벨 진행과 연동하여, 새 시스템 해금 시점에 자연스러운 학습 동기 제공
- NPC와의 관계를 의뢰 시스템으로 심화하여 마을에 생동감 부여
- 일일 목표로 매 플레이 세션마다 소소한 성취감 확보
- 도전 과제로 숙련 플레이어에게 장기 목표 제시

## 1. 퀘스트 분류 체계

SeedMind의 퀘스트는 **4개 카테고리**로 분류된다. 각 카테고리는 서로 다른 시간 스케일과 목적을 갖는다.

| 카테고리 | 영문 ID | 시간 스케일 | 목적 | 동시 활성 가능 수 |
|----------|---------|------------|------|-----------------|
| 메인 퀘스트 | `MainQuest` | 계절 (28일) | 핵심 진행 방향 제시, 시스템 해금 유도 | 계절당 3~4개 |
| NPC 의뢰 | `NPCRequest` | 3~7일 | NPC 관계 심화, 중기 보상 | 최대 3개 동시 |
| 일일 목표 | `DailyChallenge` | 1일 | 세션당 성취감, 소소한 보상 | 매일 2개 |
| 농장 도전 | `FarmChallenge` | 무기한 | 장기 목표, 마일스톤 보상 | 전체 목록 상시 열람 |

### 1.1 카테고리 간 관계

```
[메인 퀘스트]  ← 핵심 진행 (레벨/계절 연동)
      │
      ├── [NPC 의뢰]  ← 중기 콘텐츠 (NPC 관계)
      │
      ├── [일일 목표]  ← 단기 성취감 (세션 리텐션)
      │
      └── [농장 도전]  ← 장기 목표 (엔드게임)
```

**메인 퀘스트**는 게임 진행의 척추로, 계절마다 자동 부여된다. 나머지 3개 카테고리는 메인 퀘스트와 독립적으로 진행 가능하되, 메인 퀘스트 완료가 일부 NPC 의뢰/농장 도전의 해금 조건이 될 수 있다.

## 2. 퀘스트 데이터 모델

### 2.1 퀘스트 기본 구조

| 필드 | 타입 | 설명 |
|------|------|------|
| `questId` | `string` | 고유 식별자 (예: `main_spring_01`) |
| `category` | `QuestCategory` | `MainQuest` / `NPCRequest` / `DailyChallenge` / `FarmChallenge` |
| `titleKR` | `string` | 한국어 제목 |
| `descriptionKR` | `string` | 한국어 설명 |
| `giver` | `string` | 퀘스트 부여자 (NPC ID 또는 `system`) |
| `objectives` | `QuestObjective[]` | 목표 배열 (1개 이상) |
| `rewards` | `QuestReward[]` | 보상 배열 |
| `unlockCondition` | `UnlockCondition` | 해금 조건 |
| `timeLimit` | `int?` | 제한 시간 (일 단위, null이면 무기한) |
| `season` | `Season?` | 연관 계절 (null이면 전 계절 가능) |
| `isRepeatable` | `bool` | 반복 가능 여부 |

### 2.2 목표 타입 (QuestObjectiveType)

| 타입 | 영문 ID | 설명 | 예시 |
|------|---------|------|------|
| 수확 | `Harvest` | 특정 작물 N개 수확 | 감자 10개 수확 |
| 판매 | `Sell` | 특정 아이템 N개 판매 | 토마토 5개 판매 |
| 납품 | `Deliver` | 특정 아이템 N개를 NPC에게 전달 | 하나에게 딸기잼 3개 납품 |
| 가공 | `Process` | 특정 가공품 N개 제작 | 토마토 주스 3개 제작 |
| 건설 | `Build` | 특정 시설 건설 | 온실 건설 |
| 골드 | `EarnGold` | 누적/일일 골드 N 이상 획득 | 하루에 500G 이상 벌기 |
| 경작 | `Till` | 경작지 N타일 생성 | 새 경작지 10타일 만들기 |
| 물주기 | `Water` | 작물에 N회 물주기 | 오늘 15개 작물에 물주기 |
| 품질 | `QualityHarvest` | 특정 품질 이상 작물 N개 수확 | Silver 이상 작물 5개 수확 |
| 도구 | `UpgradeTool` | 특정 도구 업그레이드 수행 | 호미를 Reinforced로 업그레이드 |
| 레벨 | `ReachLevel` | 특정 레벨 도달 | 레벨 3 도달 |
| 낚시 | `Fish` | 특정 어종 N마리 포획 | 메기 3마리 낚기 |
| 복합 | `Composite` | 위 타입의 AND/OR 조합 | 토마토 수확 + 토마토 주스 제작 |

### 2.3 보상 타입 (QuestRewardType)

| 타입 | 영문 ID | 설명 |
|------|---------|------|
| 골드 | `Gold` | 골드 지급 |
| 경험치 | `XP` | 경험치 지급 |
| 아이템 | `Item` | 특정 아이템 지급 (씨앗, 비료, 가공품 등) |
| 레시피 | `Recipe` | 가공 레시피 해금 |
| 해금 | `Unlock` | 시설/기능/NPC 대화 해금 |

### 2.4 해금 조건 (UnlockCondition)

| 조건 타입 | 설명 | 예시 |
|-----------|------|------|
| `Level` | 플레이어 레벨 | `level >= 3` |
| `Season` | 특정 계절 진입 | `season == Summer` |
| `QuestComplete` | 선행 퀘스트 완료 | `main_spring_01 completed` |
| `FacilityBuilt` | 특정 시설 보유 | `has(Greenhouse)` |
| `DayOfSeason` | 계절 내 특정 일차 | `dayOfSeason >= 5` |
| `TutorialComplete` | 튜토리얼 완료 | 튜토리얼 전체 완료 후 퀘스트 시스템 활성화 |
| `ZoneUnlocked` | 특정 구역 해금 | `has(ZoneF)` |

### 2.5 진행도 추적

퀘스트 진행도는 목표별로 독립 추적된다.

```
QuestProgress {
    questId: string
    status: "Locked" | "Available" | "Active" | "Completed" | "Failed" | "Expired"
    objectiveProgress: [
        { objectiveIndex: 0, current: 7, required: 10 }
        { objectiveIndex: 1, current: 2, required: 3 }
    ]
    acceptedDay: int       // 수락한 게임 내 일차
    completedDay: int?     // 완료한 게임 내 일차
}
```

**상태 전이**:

```
[Locked] --해금 조건 충족--> [Available] --수락--> [Active]
                                                      │
                                          ┌───────────┤
                                          ↓           ↓
                                    [Completed]   [Failed/Expired]
```

- `MainQuest`와 `DailyChallenge`는 자동 수락 (Available -> Active 즉시)
- `NPCRequest`는 NPC 대화에서 수락
- `FarmChallenge`는 항상 Active (달성 시 자동 완료)
- `Failed/Expired`는 시간 제한 초과 시에만 발생 (NPC 의뢰, 일일 목표)

## 3. 계절별 메인 퀘스트

메인 퀘스트는 계절이 시작될 때 자동으로 부여되며, 해당 계절 내에 완료하는 것을 권장하지만 실패 페널티는 없다. 계절이 끝나도 미완료 메인 퀘스트는 만료되지 않고 유지된다(다만, 계절 한정 작물이 필요한 경우 다음 해 해당 계절까지 기다려야 한다).

### 3.1 봄 메인 퀘스트 (Spring Main Quests)

**전제**: 플레이어 레벨 1~3 구간, 기본 경작 시스템 학습 직후.

| ID | 제목 | 목표 | 보상 | 해금 조건 |
|----|------|------|------|-----------|
| `main_spring_01` | 봄의 첫 수확 | 감자 또는 당근 10개 수확 | 50G + 5 XP | 튜토리얼 완료 |
| `main_spring_02` | 농장 확장의 시작 | 경작지 20타일 이상 보유 | 100G + 10 XP + 고급 비료 x5 | `main_spring_01` 완료 |
| `main_spring_03` | 다양한 작물 재배 | 서로 다른 작물 2종 이상 동시 재배 | 80G + 8 XP | 레벨 2 도달 |
| `main_spring_04` | 첫 번째 출하 목표 | 총 판매 수익 500G 달성 | 150G + 13 XP + 딸기 씨앗 x3 | `main_spring_02` 완료 |

**설계 의도**: 봄 퀘스트는 튜토리얼 이후 핵심 루프 반복을 유도한다. 경작지 확장, 작물 다양화, 판매 수익 목표를 순차적으로 제시하여 "더 많이 심고, 더 넓게 확장하라"는 중기 방향성을 제공한다.

### 3.2 여름 메인 퀘스트 (Summer Main Quests)

**전제**: 플레이어 레벨 3~5 구간, 중급 작물 해금 시작.

| ID | 제목 | 목표 | 보상 | 해금 조건 |
|----|------|------|------|-----------|
| `main_summer_01` | 여름 작물 도전 | 토마토 또는 옥수수 15개 수확 | 200G + 17 XP | 여름 진입 + 레벨 3 |
| `main_summer_02` | 물탱크의 힘 | 물탱크 건설 | 300G + 22 XP | 레벨 3 + 500G 보유 |
| `main_summer_03` | 품질을 높여라 | Silver 이상 품질 작물 10개 수확 | 250G + 17 XP + 속성장 비료 x5 | 레벨 4 도달 |
| `main_summer_04` | 여름의 대수확 | 총 판매 수익 2,000G 달성 (여름 동안 누적) | 400G + 26 XP | `main_summer_01` 완료 |

**설계 의도**: 여름은 시설 건설(물탱크)과 품질 재배를 소개하는 계절이다. 물탱크 건설 퀘스트로 시설 시스템을 자연스럽게 학습시키고, 품질 퀘스트로 비료 활용을 유도한다.

### 3.3 가을 메인 퀘스트 (Autumn Main Quests)

**전제**: 플레이어 레벨 5~7 구간, 고급 작물 및 가공 시스템 진입.

| ID | 제목 | 목표 | 보상 | 해금 조건 |
|----|------|------|------|-----------|
| `main_autumn_01` | 가을의 풍성한 수확 | 호박 5개 수확 | 350G + 22 XP | 가을 진입 + 레벨 5 |
| `main_autumn_02` | 겨울 준비 | 창고 건설 + 작물 20개 이상 보관 | 500G + 32 XP | 레벨 5 |
| `main_autumn_03` | 가공의 세계 | 가공품 아무 종류 5개 제작 | 400G + 26 XP + 해바라기 씨앗 x5 | 가공소 보유 |

**설계 의도**: 가을은 겨울 대비와 가공 시스템 진입의 계절이다. 창고 건설로 작물 비축 전략을 학습시키고, 가공 퀘스트로 부가가치 창출의 재미를 소개한다.

### 3.4 겨울 메인 퀘스트 (Winter Main Quests)

**전제**: 플레이어 레벨 6~8 구간, 온실/가공 중심 플레이.

| ID | 제목 | 목표 | 보상 | 해금 조건 |
|----|------|------|------|-----------|
| `main_winter_01` | 온실의 가능성 | 온실 건설 | 600G + 32 XP | 겨울 진입 + 레벨 5 |
| `main_winter_02` | 겨울의 장인 | 가공품 10개 판매 | 500G + 26 XP | 가공소 보유 |
| `main_winter_03` | 도구의 진화 | 아무 도구 1개 Reinforced 이상으로 업그레이드 | 400G + 24 XP + 정제 강철 x2 | 레벨 6 |

**설계 의도**: 겨울은 야외 경작이 불가능하므로, 온실 건설/가공/도구 업그레이드 등 "비경작 시스템"에 집중하게 유도한다. 겨울이 지루하지 않도록 명확한 목표를 제시한다.

## 4. NPC 의뢰

NPC 의뢰는 각 NPC가 특정 조건에서 플레이어에게 제안하는 중기 퀘스트다. NPC를 방문했을 때 의뢰 아이콘(!)이 표시되며, 대화를 통해 수락한다.

### 4.1 NPC 의뢰 규칙

| 항목 | 규칙 |
|------|------|
| 동시 활성 | 최대 3개 (전체 NPC 합산) |
| 제한 시간 | 3~7일 (의뢰별 상이) |
| 실패 | 시간 초과 시 실패 처리, 페널티 없음 (동일 의뢰 재등장 가능) |
| 재등장 | 실패/완료 후 3~7일 쿨다운 뒤 다른 의뢰 등장 |
| 등장 조건 | NPC별 레벨/계절 조건 충족 시 랜덤 선택 |
| 의뢰 수락 | NPC 대화에서 "수락/거절" 선택 |
| 보상 수령 | 완료 후 NPC 재방문하여 대화로 수령 |

### 4.2 하나 (잡화 상점) 의뢰

하나는 주로 **작물 납품** 의뢰를 제공한다. 계절 작물을 활용한 요청이 중심이다.

| ID | 제목 | 목표 | 보상 | 시간 제한 | 해금 조건 |
|----|------|------|------|-----------|-----------|
| `npc_hana_01` | 신선한 감자가 필요해요 | 감자 5개 납품 | 100G + 5 XP | 5일 | 레벨 1 + 봄 |
| `npc_hana_02` | 토마토 대량 주문 | 토마토 8개 납품 | 250G + 9 XP | 7일 | 레벨 3 + 여름 |
| `npc_hana_03` | 특별한 딸기잼 | 딸기잼 3개 납품 | 400G + 14 XP + 희귀 씨앗 (수박) x1 | 7일 | 레벨 5 + 가공소 보유 |
| `npc_hana_04` | 신선한 생선이 필요해요 | 아무 생선 5마리 납품 | 150G + 7 XP | 5일 | 레벨 5 + Zone F 해금 |

**대화 예시 (npc_hana_04 수락)**:
```
[하나] "요즘 손님들이 생선 요리를 찾는데, 재고가 없어서요.
        5일 안에 생선 5마리만 가져다주실 수 있을까요?
        종류는 상관없어요!"

### 5.2 일일 목표 보상 스케일

일일 목표의 골드/XP 보상은 플레이어 레벨에 따라 스케일한다.

```
실제_보상 = 기본_보상 * (1 + (playerLevel - 1) * 0.1)
```

예시: 레벨 5 플레이어의 `daily_harvest_5` (기본 40G):
- 40 * (1 + 4 * 0.1) = 40 * 1.4 = 56G

**레벨 스케일 상한**: 레벨 10에서 최대 1.9배. 이를 초과하지 않는다.

### 7.3 퀘스트 XP와 기존 XP 시스템의 관계

퀘스트 XP는 기존 XP 소스 (-> see `docs/balance/progression-curve.md` 섹션 1.2)에 **추가 소스**로 포함된다. 기존 설계에서 XP 비중은 다음과 같다:

- 작물 수확 XP: ~55%
- 경작 활동 XP: ~15%
- 시설/진행 XP: ~12%
- 가공/판매 XP: ~3%
- 퀘스트 XP: ~10%
- 업적 XP: ~5%

퀘스트 XP는 전체 XP의 약 **11.2%(~1,010 XP)**를 차지한다. 낚시 퀘스트(낚시 도전 + 낚시 NPC 의뢰 + 일일 낚시 목표) 추가로 BAL-007 확정값(900 XP)에서 ~110 XP 증가하였다 (-> see `docs/balance/xp-integration.md` 섹션 5.2). 이를 위해 퀘스트 보상 XP 총량이 과도하지 않도록 아래 제한을 둔다.

**퀘스트 XP 예산 (연간)**:

| 소스 | 연간 총량 | 비고 |
|------|---------|------|
| 메인 퀘스트 (14개) | ~280 XP | 1년차 전체 완료 기준 |
| NPC 의뢰 (11개) | ~140 XP | 진행도에 따라 1~2년차 |
| 일일 목표 (1년차) | ~280 XP | 50% 완료율, 레벨 스케일 포함 기준 |
| 농장 도전 | ~200 XP (1년차 실현) | 초반 달성 가능 도전 ~10개 기준 |
| 낚시 도전 (6개, 1년차 실현 ~3개) | ~50 XP | Zone F 해금 후 달성 가능 도전 기준 |
| 낚시 NPC 의뢰 + 일일 낚시 목표 (1년차) | ~60 XP | NPC 의뢰 7 XP x2회 + 일일 낚시 목표 2 XP x ~23회 |

[RESOLVED -- BAL-007 + CON-010] 퀘스트/업적 XP 포함 통합 시뮬레이션 완료. 낚시 퀘스트 추가 반영 후 확정된 통합 배분(수확 ~76%, 퀘스트 ~12%, 업적 ~12%)과 1년차 레벨 8 중반 목표가 확정되었다 (-> see `docs/balance/xp-integration.md` 섹션 5).

│  ── NPC 의뢰 ──                                     │
│  [하나] 신선한 감자가 필요해요  [3/5] ██████░░░░    │
│         남은 기한: 3일                              │
│                                                     │
│  ── 일일 목표 ──                                    │
│  [*] 부지런한 농부         [12/15] ████████████░░░  │
│  [*] 오늘의 수확           [3/5] ██████░░░░         │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### 8.2 퀘스트 상태 아이콘

| 아이콘 | 의미 |
|--------|------|
| `[!]` | 활성 진행 중 (메인 퀘스트) |
| `[*]` | 활성 진행 중 (일일 목표) |
| `[V]` | 완료 (보상 수령 가능) |
| `[X]` | 실패/만료 |
| 잠금 아이콘 | 해금 조건 미충족 |

### 8.3 HUD 퀘스트 추적기

메인 화면 우측에 현재 추적 중인 퀘스트 1~2개의 진행도를 간략히 표시한다.

```
┌──────────────────────────────────────────┐
│ [시간/날짜]  [계절]  [날씨]    [골드: 0G] │
│                                          │
│                              ┌─────────┐ │
│              (게임 화면)     │봄의 첫수확│ │
│                              │ 7/10    │ │
│                              └─────────┘ │
│                                          │
│ [도구바: 호미 | 물뿌리개 | 씨앗 | 낫]    │
│ [레벨: 1]  [경험치 바]                    │
└──────────────────────────────────────────┘
```

**추적 퀘스트 설정**: 퀘스트 로그에서 원하는 퀘스트를 "추적" 설정하면 HUD에 표시. 기본적으로 메인 퀘스트 1개가 자동 추적된다.

### 8.4 진행도 표시 규칙

| 목표 타입 | 표시 형식 | 예시 |
|-----------|----------|------|
| 수량 기반 (수확, 납품 등) | `현재/목표` + 진행 바 | `7/10 ████████░░` |
| 이진 기반 (건설, 레벨 도달) | 체크박스 | `[ ] 온실 건설` → `[V] 온실 건설` |
| 골드 기반 | `현재G/목표G` + 진행 바 | `350G/500G ███████░░░` |
| 복합 목표 | 각 하위 목표별 표시 | 목표 1: [V], 목표 2: [3/5] |

### 8.5 완료 연출

| 이벤트 | 연출 |
|--------|------|
| 목표 달성 (진행도 갱신) | 화면 우측 플로팅 텍스트: "+1 감자 수확 (7/10)" |
| 퀘스트 완료 | 화면 중앙 배너: "퀘스트 완료! [봄의 첫 수확]", 보상 아이콘 표시, 효과음 |
| 농장 도전 달성 | 전용 팝업: 도전 이름 + 보상 + "달성!" 텍스트, 별 이펙트 |
| 일일 목표 완료 | 작은 배너: "일일 목표 완료! +40G +2 XP" |
| NPC 의뢰 완료 | NPC 머리 위 아이콘 변경 (! -> V), 대화에서 보상 수령 |

### 8.6 알림 시스템

| 상황 | 알림 |
|------|------|
| 새 퀘스트 해금 | 화면 상단 알림 바: "새 퀘스트: [제목]" |
| 새 NPC 의뢰 등장 | NPC 위치에 느낌표(!) 아이콘 |
| 의뢰 기한 임박 (1일 전) | 화면 상단 경고: "[제목] 기한이 내일까지입니다!" |
| 일일 목표 갱신 (매일 아침) | 아침 기상 시 팝업: "오늘의 목표 2개가 설정되었습니다" |

## 9. 시스템 연동 상세

### 9.1 튜토리얼과의 관계

퀘스트 시스템은 튜토리얼 완료 후 활성화된다 (-> see `docs/systems/tutorial-system.md`). 튜토리얼이 "핵심 루프 1회전 학습"을 담당하고, 메인 퀘스트가 "핵심 루프 반복 + 확장"을 담당한다.

- 튜토리얼 완료 = `main_spring_01` 해금 트리거
- 튜토리얼 중에는 퀘스트 로그 UI 비활성

### 9.2 세이브/로드와의 관계

퀘스트 진행도는 세이브 데이터에 포함된다 (-> see `docs/systems/save-load-architecture.md`).

저장 대상:
- 모든 퀘스트의 현재 상태 (`QuestProgress` 배열)
- 일일 목표 풀의 금일 선택 결과
- NPC 의뢰 쿨다운 타이머
- 농장 도전 달성 기록 (영구)
- 누적 통계 (총 수확 수, 총 판매액 등 — 농장 도전 조건 추적용)

### 9.3 경제 시스템과의 관계

퀘스트 골드 보상은 경제 시스템의 골드 획득 경로에 추가된다 (-> see `docs/systems/economy-system.md` 섹션 1.3).

**골드 인플레이션 방지 규칙**:
- 메인 퀘스트 골드 보상 합계(계절당) < 해당 계절 작물 판매 예상 수익의 30%
- NPC 의뢰 골드 보상 < 납품 아이템의 시장가 x2 (납품 기회비용 고려)
- 일일 목표 골드 보상은 일일 예상 수입의 10~20% 수준

봄 기준 검증:
- 봄 작물 판매 예상 수익: 약 1,500~2,000G (-> see `docs/balance/progression-curve.md` 섹션 3)
- 봄 메인 퀘스트 골드 보상 합계: 50 + 100 + 80 + 150 = 380G (예상 수익의 ~22%) -- 적정

## Risks

- [RISK] 퀘스트 보상 골드가 작물 판매 수익 대비 과도할 경우, "농사보다 퀘스트가 효율적"인 상황이 발생하여 핵심 루프를 훼손할 수 있다. 섹션 9.3의 인플레이션 방지 규칙을 반드시 준수해야 한다.
- [RISK] 일일 목표가 플레이어의 자유도를 제한할 수 있다. "오늘 하고 싶은 것"과 "일일 목표"가 충돌하면 스트레스 요인이 된다. 일일 목표 실패에 페널티가 없으므로 압박감은 제한적이나, 보상이 매력적이면 "안 하면 손해"라는 압박이 발생한다.
- [RISK] NPC 의뢰의 납품 아이템이 현재 계절에서 구할 수 없는 경우(계절 불일치 버그), 의뢰를 수락했으나 완료 불가능한 상황 발생. 의뢰 등장 조건에 계절 필터를 반드시 적용해야 한다.
- [RISK] 농장 도전 보상 총량이 과도하면 XP 인플레이션 유발. 낚시 도전 6종 추가(총 XP 140)를 포함하여 모든 도전을 달성할 경우 추가 XP ~3,890. 이는 조정된 총 누적 XP 9,029의 약 43%에 해당하므로, 도전 보상 XP를 30% 이하로 억제하는 조정이 필요할 수 있다.
- [RISK] 낚시 퀘스트(NPC 의뢰 + 일일 목표 + 농장 도전)의 XP가 기존 퀘스트 XP 예산(~900)에 ~110 XP를 추가한다. 퀘스트 XP 비중이 10% -> ~11.2%로 소폭 증가하나 허용 범위 내. 단, 낚시 업적 XP(390)와 합산 시 전체 영향 재검증 필요.
