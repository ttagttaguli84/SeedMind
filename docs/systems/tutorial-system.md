## Context

이 문서는 SeedMind의 튜토리얼 및 온보딩 시스템을 상세히 기술한다. 플레이어가 게임을 처음 시작했을 때 핵심 메카닉(경작, 성장, 수확, 판매, 해금)을 자연스럽게 학습하는 전체 흐름을 설계한다. 마스터 디자인 문서(`docs/design.md`)의 핵심 게임 루프와 각 시스템 문서에서 정의된 메카닉을 기반으로, "어떤 순서로, 어떤 방식으로, 누구를 통해" 가르칠 것인지를 명세한다.

**설계 목표**: 플레이어가 "지시받는 느낌" 없이 자연스럽게 게임을 배운다. NPC의 대화와 상황별 힌트를 통해 학습이 플레이의 일부가 되도록 한다. 동시에 숙련 플레이어는 빠르게 스킵하고 자유 플레이에 진입할 수 있어야 한다.

**본 문서가 canonical인 데이터**:
- 튜토리얼 단계 정의 (트리거, 완료 조건, 보상)

- 튜토리얼 NPC 연계 구조

## 1. 온보딩 철학

### 1.1 핵심 원칙

| 원칙 | 설명 | 적용 |
|------|------|------|
| **Do, Don't Tell** | 텍스트 설명보다 직접 행동하며 배운다 | 모든 튜토리얼 단계에 "직접 해보기" 목표가 존재 |
| **컨텍스트 기반 안내** | 플레이어가 특정 상황에 처했을 때만 힌트를 표시 | 물이 부족한 작물 옆에서만 물주기 안내 등장 |
| **NPC를 통한 전달** | 시스템 메시지가 아니라 NPC 대화로 정보를 전달 | 하나가 씨앗 사용법을 알려주고, 철수가 도구를 설명 |
| **단계적 복잡도** | 한 번에 하나의 메카닉만 소개 | 비료/토양 품질은 튜토리얼 범위 밖 (레벨 해금 시 별도 안내) |
| **실패 허용** | 작물이 죽어도 게임이 끝나지 않음을 체감시킴 | 첫 작물 고사 시 위로 대사 + 무료 씨앗 제공 |
| **스킵 자유** | 경험 있는 플레이어는 즉시 자유 플레이 가능 | 첫 대화에서 "건너뛰기" 선택지 제공 |

### 1.2 튜토리얼 범위

튜토리얼은 플레이어가 **핵심 게임 루프 1회전**을 완수하는 것을 목표로 한다.

```
[땅 경작] → [씨앗 심기] → [물주기] → [성장 대기] → [수확] → [판매] → [새 씨앗 구매]
```

**튜토리얼에 포함되는 시스템**:
- 이동 및 기본 조작
- 도구 선택 및 사용 (호미, 물뿌리개, 씨앗봉투, 낫)
- 인벤토리 열기/확인
- 상점 방문 및 구매/판매
- 시간 흐름 인지 (하루/수면)

**튜토리얼에 포함되지 않는 시스템** (레벨 해금 시 컨텍스트 힌트로 안내):
- 비료 시스템 (레벨 3+ 해금, -> see 섹션 7.2)
- 토양 품질/연작 피해 (레벨 3+ 표시 시작, -> see `docs/systems/farming-system.md` 섹션 6)
- 도구 업그레이드 (레벨 3+ 해금, -> see `docs/systems/tool-upgrade.md` 섹션 1.1)
- 시설 건설 (레벨별 해금, -> see `docs/design.md` 섹션 4.6)
- 가공 시스템 (레벨 7+ 해금)
- 날씨 영향 상세 (첫 비 이벤트 시 안내)

### 1.3 학습 곡선 설계

```
복잡도
  ^
  |                                          ┌─ 가공, 교잡
  |                                     ┌────┘
  |                               ┌─────┘  시설 건설, 도구 업그레이드
  |                          ┌────┘
  |                     ┌────┘  비료, 토양, 날씨 영향
  |               ┌─────┘
  |          ┌────┘  상점 거래, 인벤토리 관리
  |     ┌────┘
  |─────┘  이동, 경작, 씨앗, 물, 수확
  |
  └──────────────────────────────────────────> 시간
    [튜토리얼]  [초반 자유]  [중반 확장]  [후반 최적화]
     Day 1~3     Day 4~28    Season 2~3   Season 4+
```

## 2. 튜토리얼 플로우 (Tutorial Flow)

### 2.0 플로우 개요

튜토리얼은 12단계로 구성되며, 게임 첫 1~3일(실시간 10~30분) 내에 완료되도록 설계한다. 각 단계는 명확한 트리거(시작 조건), 목표, 완료 조건, 보상을 갖는다.

```
[S01 도착]→[S02 이동]→[S03 하나]→[S04 경작]→[S05 씨앗]→[S06 물주기]
                                                              ↓
[S12 새싹]←[S11 재투자]←[S10 판매]←[S09 수확]←[S08 성장]←[S07 수면]
```

### 2.1 단계 상세

---

#### S01. 농장 도착 (Arrival)

| 항목 | 내용 |
|------|------|
| **트리거** | 새 게임 시작 |
| **NPC** | 없음 (내레이션) |
| **목표** | 게임 세계에 대한 첫인상 형성 |
| **연출** | 카메라가 농장 전경을 천천히 패닝한 뒤, 플레이어 캐릭터 위치로 줌인. 화면 하단에 "SeedMind Farm" 타이틀 텍스트 페이드인 |
| **완료 조건** | 카메라 연출 종료 (자동, 약 5초) 또는 아무 키 입력으로 스킵 |
| **보상** | 없음 |
| **가이드 UI** | 없음 (순수 시네마틱) |

**스킵 분기점**: 타이틀 연출 중 ESC를 누르면 "튜토리얼을 건너뛸까요?" 팝업 표시. "예"를 선택하면 초기 아이템을 인벤토리에 지급하고 자유 플레이로 전환 (-> see 섹션 4.1).

---

#### S02. 첫걸음 — 이동 배우기 (First Steps)

| 항목 | 내용 |
|------|------|
| **트리거** | S01 완료 직후 |
| **NPC** | 없음 |
| **목표** | WASD 이동과 카메라 조작 학습 |
| **가이드 UI** | 화면 중앙 하단에 WASD 키 아이콘 + "이동하기" 텍스트. 마우스 휠 아이콘 + "줌 인/아웃" |
| **완료 조건** | 플레이어가 WASD 중 2개 이상의 방향으로 이동 (총 이동 거리 5타일 이상) |
| **보상** | 없음 |
| **실패 안전장치** | 15초간 입력이 없으면 키 아이콘이 펄스 애니메이션으로 강조 |

---

#### S03. 하나와의 만남 — 상점 방문 (Meet Hana)

| 항목 | 내용 |
|------|------|
| **트리거** | S02 완료 후 |
| **NPC** | 하나 (npc_hana) |
| **목표** | NPC 상호작용 방법 학습 + 초기 아이템 수령 |
| **가이드 UI** | 하나 위치에 노란색 느낌표(!) 마커 표시. 화면 상단에 "마을의 하나를 찾아가 보세요" 퀘스트 텍스트. 미니맵에 하나 위치 핀 |
| **대화 내용** | 아래 스크립트 참조 |
| **완료 조건** | 하나와의 대화 완료 + 초기 아이템 수령 확인 |
| **보상** | 감자 씨앗 5개 (인벤토리 직접 지급) |
| **실패 안전장치** | 30초간 하나에게 접근하지 않으면 화살표 가이드 추가 |

**S03 대화 스크립트**:

```
[하나] "안녕! 새로 농장을 맡게 된 분이죠?
        저는 하나, 이 마을의 잡화 상점을 운영하고 있어요."

[하나] "처음이니까 제가 감자 씨앗을 좀 드릴게요.
        농장에 가서 한번 심어 보세요!"

        → [감자 씨앗 x5 획득!] (화면 중앙 아이템 획득 연출)

[하나] "참, 도구 사용법은 아시죠?
        숫자 키 1~5로 도구를 바꿀 수 있어요.
        먼저 호미(1번)로 땅을 갈아야 씨앗을 심을 수 있답니다."

        → [도구바 하이라이트: 호미 슬롯 깜빡임]
```

#### S04. 땅 갈기 — 호미 사용 (Till the Soil)

| 항목 | 내용 |
|------|------|
| **트리거** | S03 완료 후, 플레이어가 농장 구역에 진입 |
| **NPC** | 없음 (시스템 안내) |
| **목표** | 호미 도구로 Empty 타일을 Tilled 상태로 전환하는 법 학습 |
| **가이드 UI** | (1) 도구바에서 호미 슬롯(1번) 하이라이트. (2) 농장의 특정 타일 3개에 점선 원 마커 표시 ("여기를 갈아 보세요"). (3) 화면 상단: "1번 키로 호미를 선택하고 좌클릭으로 땅을 갈아 보세요" |
| **완료 조건** | 타일 3개 이상을 Tilled 상태로 전환 |
| **보상** | 없음 (즉각적 시각/청각 피드백이 보상) |
| **실패 안전장치** | 잘못된 타일에 호미 사용 시 "빈 땅 위에서 사용하세요" 말풍선 |

**튜닝 파라미터**:

| 파라미터 | 값 | 설명 |
|----------|-----|------|
| `tutorial_tillTarget` | 3 | 경작 목표 타일 수 |
| `tutorial_tillMarkerCount` | 3 | 가이드 마커 표시 타일 수 |

#### S05. 씨앗 심기 — 첫 파종 (Plant Seeds)

| 항목 | 내용 |
|------|------|
| **트리거** | S04 완료 직후 |
| **NPC** | 없음 (시스템 안내) |
| **목표** | 씨앗봉투 도구로 Tilled 타일에 씨앗을 심는 법 학습 |
| **가이드 UI** | (1) 도구바에서 씨앗봉투 슬롯(3번) 하이라이트. (2) 경작된 타일에 씨앗 아이콘 가이드. (3) 화면 상단: "3번 키로 씨앗을 선택하고, 경작한 땅에 심어 보세요" |
| **완료 조건** | 씨앗 3개 이상 심기 (타일이 Planted 상태로 전환) |
| **보상** | 없음 |
| **실패 안전장치** | 경작되지 않은 타일에 씨앗 사용 시도 시 "먼저 호미로 땅을 갈아야 해요" 말풍선 |

#### S06. 물주기 — 성장의 시작 (Water the Crops)

| 항목 | 내용 |
|------|------|
| **트리거** | S05 완료 직후 |
| **NPC** | 없음 (시스템 안내) |
| **목표** | 물뿌리개로 Planted 타일에 물을 주는 법 학습 + 물의 중요성 인지 |
| **가이드 UI** | (1) 도구바에서 물뿌리개 슬롯(2번) 하이라이트. (2) 심어진 타일에 물방울 아이콘 표시. (3) 화면 상단: "2번 키로 물뿌리개를 선택하고 물을 주세요. 물을 줘야 작물이 자라요!" |
| **완료 조건** | 심은 씨앗 전부에 물주기 완료 (모든 Planted 타일 -> Watered) |
| **보상** | 없음 |
| **추가 안내** | 물주기 완료 후 말풍선: "잘했어요! 매일 물을 줘야 작물이 자란답니다. 내일도 잊지 마세요." |

#### S07. 하루의 끝 — 수면과 시간 (End of Day)

| 항목 | 내용 |
|------|------|

#### S08. 성장 확인 — 새싹 발견 (Growth Check)

| 항목 | 내용 |
|------|------|
| **트리거** | Day 2 아침, 플레이어가 농장에 접근 |

#### S09. 첫 수확 — 수확의 기쁨 (First Harvest)

| 항목 | 내용 |

## 3. UI/HUD 가이드 요소

### 3.1 가이드 요소 유형

| 요소 | 영문 키 | 사용 상황 | 시각적 표현 |
|------|---------|-----------|------------|
| 방향 화살표 | `ArrowGuide` | 특정 위치로 이동 유도 | 반투명 노란색 화살표, 대상까지 부드럽게 가리킴. 플레이어와 대상 간 거리에 따라 크기 변동 |
| 슬롯 하이라이트 | `SlotHighlight` | 도구바에서 특정 도구 선택 유도 | 해당 슬롯 테두리가 노란색으로 깜빡임 (0.5초 주기) |
| 타일 마커 | `TileMarker` | 특정 타일에서 행동 유도 | 타일 위에 점선 원 + 도구 아이콘 (예: 호미 아이콘이면 "여기를 갈아라") |
| 말풍선 힌트 | `SpeechBubble` | 짧은 텍스트 정보 전달 | 캐릭터/NPC 머리 위 만화 스타일 말풍선. 배경: 반투명 흰색, 테두리: 짙은 갈색 |
| 퀘스트 텍스트 | `QuestText` | 현재 목표 표시 | 화면 상단 중앙, 반투명 배경 위에 흰색 텍스트. 체크마크 애니메이션으로 완료 표시 |
| NPC 마커 | `NpcMarker` | NPC 위치 + 상호작용 가능 표시 | NPC 머리 위 노란색 느낌표(!). 상호작용 범위 진입 시 "E" 키 아이콘으로 변경 |
| 키 프롬프트 | `KeyPrompt` | 특정 키 입력 유도 | 화면 하단 중앙에 키보드 키 아이콘 + 동작 설명 (예: "[E] 대화하기") |

### 3.2 가이드 표시 규칙

| 규칙 | 설명 |
|------|------|
| 중첩 금지 | 한 화면에 동시에 표시되는 가이드 요소는 최대 3개. 우선순위: 퀘스트 텍스트 > NPC 마커 > 타일 마커 > 방향 화살표 |
| 자동 소멸 | 완료 조건 달성 시 가이드 요소는 0.3초 페이드아웃으로 사라짐 |
| 비활성 투명도 | 비관련 UI 요소는 튜토리얼 단계 중 50% 투명도로 표시 (포커스 유도) |
| 반복 표시 | 가이드를 닫은 후 10초 이상 목표를 달성하지 못하면 다시 표시 |
| 화면 밖 대상 | 대상이 화면 밖에 있으면 화면 가장자리에 방향 지시자(삼각형) 표시 |

### 3.3 튜토리얼 전용 UI 오버레이

튜토리얼 활성 상태에서는 HUD에 추가 요소가 표시된다:

```
┌──────────────────────────────────────────────┐
│ [시간/날짜]  [계절]  [날씨]      [골드: 500G] │
│                                              │
│    ┌──────────────────────────┐              │
│    │  "호미로 땅을 3칸        │  ← 퀘스트    │
│    │   갈아 보세요 (1/3)"    │     텍스트    │
│    └──────────────────────────┘              │
│                                              │
│              (게임 화면)                      │
│                                              │
│ [▸도구바: *호미* | 물뿌리개 | 씨앗 | 낫]     │
│             ↑ 슬롯 하이라이트                │
│ [레벨: 1]  [경험치 바]                        │
│                                              │
│ ┌────────────────────────────────────────┐   │
│ │ [WASD] 이동  [좌클릭] 도구 사용        │   │
```

## 5. NPC 연계

### 5.1 NPC 역할 분담

각 NPC는 특정 시스템의 "안내자" 역할을 맡는다. 플레이어가 해당 시스템에 처음 접근할 때 NPC가 자연스럽게 설명을 제공한다.

| NPC | 담당 시스템 | 안내 시점 | 안내 방식 |
|-----|-----------|-----------|-----------|
| 하나 (Hana) | 경작 기본, 씨앗/작물, 상점 거래 | 튜토리얼 S03, S10, S11 + 레벨 해금 시 | 밝은 어조의 직접 대화. 계절별 작물 추천 |
| 철수 (Cheolsu) | 도구 업그레이드, 재료 | 레벨 3 도달 후 첫 대장간 방문 시 | 과묵한 어조의 짧은 설명. 도구 관리 팁 |
| 목이 (Moki) | 시설 건설, 농장 확장 | 레벨 3 도달 후 첫 목공소 방문 시 | 따뜻한 어조의 건설 안내. 시설 효과 설명 |

### 5.2 하나 — 초반 안내자 (Primary Guide)

하나는 튜토리얼의 주요 안내자이며, 게임 전반에 걸쳐 가장 자주 접촉하는 NPC이다.

**튜토리얼 대사 트리거**:

| 조건 | 대사 | 1회성 |
|------|------|:-----:|
| 첫 만남 (S03) | 씨앗 지급 + 도구 설명 | O |
| 첫 수확 후 방문 | "첫 수확 축하! 출하함이나 여기서 팔 수 있어요" | O |
| 레벨 2 후 방문 | "토마토 씨앗이 들어왔어요! 여름에 심어 보세요" | O |
| 레벨 3 후 방문 | "고급 비료도 준비했어요. 한번 써 보세요" | O |
| 처음으로 작물 고사 발견 후 | "앗, 작물이 시들었네요... 매일 물을 줘야 해요. 씨앗 하나 더 드릴게요" (감자 씨앗 1개 추가 지급) | O |

### 5.5 NPC 대사 우선순위 시스템

NPC에게 여러 특수 대사 조건이 동시에 충족될 경우, 다음 우선순위로 1개만 표시한다:

| 우선순위 | 대사 유형 | 예시 |
|----------|-----------|------|
| 1 (최고) | 튜토리얼 스텝 대사 | S03, S10 등 |
| 2 | 진행도 마일스톤 대사 | 레벨 해금, 첫 수확 등 |
| 3 | 계절 전환 대사 | "봄이에요! 감자가 잘 나가요" |
| 4 | 일반 인사 | "안녕하세요! 오늘은 뭘 찾으세요?" |

표시되지 않은 상위 대사는 큐에 보관되어, 다음 방문 시 표시된다. 큐에 2개 이상 쌓이면 가장 오래된 것부터 표시한다.

## 6. 진행 체크포인트

### 6.1 체크포인트 정의

튜토리얼 진행을 추적하는 주요 체크포인트와 보상 구조이다.

| 체크포인트 ID | 이름 | 조건 | 보상 |
|-------------|------|------|------|
| `CP_TILL` | 첫 경작 | 타일 1개 Tilled 전환 | 없음 (즉각 피드백) |
| `CP_PLANT` | 첫 파종 | 씨앗 1개 심기 | 없음 |
| `CP_WATER` | 첫 물주기 | 물 1회 주기 | 없음 |
| `CP_HARVEST` | 첫 수확 | 작물 1개 수확 | XP 보너스 (-> see `docs/balance/progression-curve.md` 섹션 1.2.4) |
| `CP_SELL` | 첫 판매 | 작물 1개 판매 | 없음 |
| `CP_BUY` | 첫 구매 | 상점에서 아이템 구매 | 없음 |
| `CP_TUTORIAL_DONE` | 튜토리얼 완료 | S12 도달 | 당근 씨앗 3개 + "새싹 농부" 칭호 |

### 6.2 보상 설계 원칙

| 원칙 | 적용 |
|------|------|
| 과도한 보상 금지 | 튜토리얼 보상이 초반 경제를 왜곡하지 않도록 최소화. 씨앗 수준만 지급 |
| 경험치 보상은 밸런스 연동 | 첫 수확 XP는 `progression-curve.md`에 정의된 값만 사용 |
| 칭호는 수집 동기 | "새싹 농부"는 첫 번째 칭호로, 이후 칭호 수집 시스템의 시작점 |
| 물질 보상 < 체험 보상 | 골드/아이템보다 연출(팡파레, 파티클)과 NPC 축하 대사가 더 큰 보상감 |

### 6.3 튜토리얼 중 보호 규칙

초보 플레이어가 튜토리얼 중 불이익을 받지 않도록 다음 보호 규칙을 적용한다:

| 규칙 | 설명 | 해제 시점 |
|------|------|-----------|
| 기절 페널티 면제 | 24:00 기절 시 골드 차감 없음 | 튜토리얼 완료(S12) 후 |
| 작물 고사 보상 | 첫 고사 발견 시 감자 씨앗 1개 무료 지급 | 1회성 |
| 에너지 부족 경고 강화 | 에너지 30 이하에서 추가 시각 경고 | 튜토리얼 완료 후 기본 20으로 복원 |
| 계절 전환 유예 | 튜토리얼 진행 중 계절이 전환되지 않음 (Day 28 초과 시 자동 연장) | 튜토리얼 완료 후 정상 진행 |

[OPEN] 계절 전환 유예 규칙이 플레이 시간이 극도로 긴 플레이어에게 어뷰징 가능성이 있는지 검토 필요. 유예 기간 상한(예: Day 35)을 설정하는 방안 고려.

## 7. 컨텍스트 힌트 시스템 (Context Hint System)

### 7.1 개요

컨텍스트 힌트는 튜토리얼 완료 후에도 게임 전반에 걸쳐 작동하는 상황별 자동 팁 시스템이다. 플레이어가 특정 상황에 처음 직면할 때 한 번만 표시되며, 강제성이 없고 즉시 닫을 수 있다.

**표시 형식**: 화면 우측 하단에 슬라이드인 되는 알림 패널. NPC 초상화(관련 NPC가 있는 경우) + 짧은 텍스트 (2줄 이내). 5초 후 자동 페이드아웃 또는 클릭으로 즉시 닫기.

```
┌─────────────────────────────────┐
│ [하나 초상화]  "비가 오면 물을  │
│                주지 않아도 돼요!"│
│                    [X 닫기]     │
└─────────────────────────────────┘
```

## Risks

1. [RISK] **튜토리얼 피로감**: 12단계가 경험 많은 플레이어에게 지루할 수 있음. 스킵 시스템(섹션 4)으로 완화하나, 스킵 버튼의 가시성이 부족하면 이탈 위험 존재.
2. [RISK] **성장 대기 구간의 공백**: S06(물주기) 완료 후 S09(수확)까지 작물 성장을 기다려야 하는 시간이 발생. 이 구간에서 플레이어가 "할 일이 없다"고 느낄 수 있음. 대안: 대기 시간 동안 마을 탐험/NPC 대화를 유도하는 선택적 서브 퀘스트 검토.
3. [RISK] **NPC 대사 분량**: 튜토리얼 + 컨텍스트 힌트 + 일반 대사를 합산하면 하나의 대사 분량이 상당히 많아짐. 로컬라이제이션이나 음성 지원 시 부담이 될 수 있음.
4. [RISK] **보호 규칙의 부작용**: 기절 페널티 면제 + 계절 유예가 튜토리얼 미완료 시 장기간 유지되면 밸런스를 왜곡할 수 있음. `tutorial_seasonFreezeMaxDay` 상한으로 일부 완화.
