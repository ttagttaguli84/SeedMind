## 2. 분석 기준

### 2.1 기준선 정의

여행 상인 아이템의 가격 적정성을 판단하기 위해, 게임 내 주요 수익원의 일일 효율을 기준선으로 설정한다.

| 기준선 | 수치 | 출처 |
|--------|------|------|
| 초반 작물 일일 수익 (감자) | 5.0G/일 | (-> see `docs/balance/crop-economy.md` 섹션 1.2) |
| 중반 작물 일일 수익 (옥수수) | 7.1G/일 | (-> see `docs/balance/crop-economy.md` 섹션 1.2) |
| 후반 작물 일일 수익 (수박) | 14.6G/일 | (-> see `docs/balance/crop-economy.md` 섹션 1.2) |
| 딸기 (다중 수확, 최고 효율) | 21.4G/일 | (-> see `docs/balance/crop-economy.md` 섹션 1.3) |
| 닭 일일 순수익 (일반) | 25G/일 | (-> see `docs/balance/livestock-economy.md` 섹션 1.2) |
| 1년차 봄 종료 잔고 | ~2,345G | (-> see `docs/balance/progression-curve.md` 섹션 3.2) |
| 1년차 여름 종료 잔고 | ~2,885G | (-> see `docs/balance/progression-curve.md` 섹션 3.2) |
| 1년차 겨울 종료 잔고 | ~8,270G | (-> see `docs/balance/progression-curve.md` 섹션 3.2) |
| 게임 시작 골드 | 500G | (-> see `docs/systems/economy-system.md` 섹션 1.2) |

### 2.2 가격 적정성 판단 원칙

1. **회수 기간 (Payback Period)**: 아이템 투자금을 회수하는 데 걸리는 게임 내 일수. 1시즌(28일) 이내 회수가 바람직하다.
2. **잔고 대비 부담도**: 아이템 가격이 해당 시점 예상 잔고의 몇 %를 차지하는지. 5~15%가 "고민하게 만드는" 적정 범위이다. 20% 이상이면 너무 비싸고, 3% 이하면 고민 없이 구매하게 된다.
3. **대안 비교**: 같은 효과를 달성할 수 있는 일반 아이템/행동이 있다면, 그 대안 대비 편의성 프리미엄이 1.5~2.5배 범위여야 한다.
4. **희소성 보정**: 여행 상인 전용 아이템(다른 경로로 획득 불가)은 프리미엄이 높아도 정당화된다. 대안이 있는 아이템은 프리미엄이 낮아야 한다.

## 3. 아이템별 ROI 분석

### 3.1 만능 비료 (Universal Fertilizer) -- 150G

**기본 데이터** (-> see `docs/content/npcs.md` 섹션 6.4):
- 가격: 150G
- 효과: 품질 보정 +15% + 성장 속도 +10%
- 여행 상인 전용 (독점)

**비교 대상**: 고급 비료(50G, 품질 +30%) + 속성장 비료(40G, 성장 +20%). 각각의 전문 비료보다 효과가 절반 수준이지만, 한 타일에 두 가지 효과를 동시에 적용하는 편의성이 핵심 가치이다.

**ROI 분석 -- 수박에 적용 시** (가장 유리한 케이스):

```
만능 비료 적용 시 (Normal 토양):
  품질 보정: 기본점수 0.1 + 만능 비료 0.15 = 0.25
  -> 품질 확률: Normal 17%, Silver 83%, Gold 0%, Iridium 0%
  기대 배수: 0.17 x 1.0 + 0.83 x 1.25 = 1.21
  수박 기대 판매가: 350G x 1.21 = 423.5G

  성장 단축: 수박 12일, 주계절 x1.1, 만능비료 +10% (x1.1)
  종합 배수: 1.1 x 1.1 = 1.21
  실제 성장일수: ceil(12 / 1.21) = 10일
  시즌 회전수: floor(28 / 10) = 2회전

비교 -- 비료 없음 (수박):
  기대 배수: 1.08 (-> see `docs/balance/crop-economy.md` 섹션 3.2)
  기대 판매가: 350G x 1.08 = 378G
  성장일수: ceil(12 / 1.1) = 11일 -> 2회전

비교 -- 기본 비료(20G, 성장 x1.25):
  회전수: 3회전 (9일)
  시즌 순이익: 3 x 350 - 3 x 175 - 3 x 20 = 465G

만능 비료 시즌 순이익 (수박):
  2회전 x 423.5G - 2 x 175G - 2 x 150G = 847G - 350G - 300G = 197G

비료 없음 시즌 순이익 (수박):
  2회전 x 378G - 2 x 175G = 756G - 350G = 406G
```

**핵심 발견**: 만능 비료(150G)를 수박에 적용하면 시즌 순이익이 오히려 **감소**한다(406G -> 197G). 성장 속도 +10%가 추가 회전을 만들어내지 못하면서, 150G의 비용만 소모된다. 기본 비료(20G)가 465G로 압도적으로 우월하다.

**만능 비료가 추가 회전을 만들 수 있는 작물**:

| 작물 | 만능 비료 적용 일수 | 비료 없음(주계절) | 추가 회전? | 비고 |
|------|-------------------|-----------------|-----------|------|
| 감자 (3일) | ceil(3/1.21)=3일 | 3일 | 없음 | 단축 없음 |
| 당근 (3일) | 3일 | 3일 | 없음 | 단축 없음 |
| 토마토 (5일) | ceil(5/1.21)=5일 | 5일 | 없음 | 단축 없음 |
| 옥수수 (7일) | ceil(7/1.21)=6일 | 7일 | **있음** (4->4회전) | 효과 미미 |
| 호박 (10일) | ceil(10/1.21)=9일 | 10일 | **있음** (2->3회전) | 가장 유리 |
| 해바라기 (8일) | ceil(8/1.21)=7일 | 8일 | **있음** (3->4회전) | |
| 수박 (12일) | ceil(12/1.21)=10일 | 11일 | 없음 | |

**호박에 적용 시 (최적 케이스)**:

```
만능 비료 + 호박 (주계절 가을):
  성장일수: ceil(10 / 1.21) = 9일 -> 3회전
  기대 배수: 1.21 (품질 보정 반영)
  시즌 순이익: 3 x (200 x 1.21) - 3 x 100 - 3 x 150
           = 726G - 300G - 450G = -24G (적자!)

비료 없음 + 호박:
  2회전 x (200 x 1.08) - 2 x 100 = 432G - 200G = 232G
```

**[RISK] 만능 비료의 가성비 심각 부족**: 150G라는 높은 가격에 비해 효과(품질 +15%, 성장 +10%)가 너무 약하다. 어떤 작물에 적용해도 비료 없음보다 순이익이 낮거나 적자이다. 현재 가격으로는 구매 가치가 전혀 없다.

**권장 조정**: 섹션 5.1 참조.

### 3.2 비계절 씨앗 (Off-Season Seeds) -- 정가 x2.0

**기본 데이터** (-> see `docs/content/npcs.md` 섹션 6.3):
- 가격: 해당 씨앗 정가의 2배
- 효과: 다음 계절 작물을 미리 구매 가능
- 등장 확률: 30%

**ROI 분석 -- 시간 이점의 가치**:

비계절 씨앗의 가치는 "시즌 1일차부터 심을 수 있다"는 시간 이점에 있다. 하나 상점은 해당 계절이 시작되면 자동으로 그 계절 씨앗을 판매하므로, 비계절 씨앗 없이도 Day 1에 구매-파종이 가능하다. 다만 "전 계절 마지막 주에 미리 구매하여 Day 1 오전부터 바로 심기"가 가능해진다.

```
시간 이점 시나리오:
  일반: 새 계절 Day 1 -> 상점 방문(이동 시간) -> 구매 -> 밭으로 이동 -> 파종
  비계절 씨앗: 전 계절 마지막 주에 구매 완료 -> Day 1 시작과 동시에 파종

  실제 시간 이점: 게임 내 약 반나절(상점 왕복 + 구매 시간)
  -> 일일 수익률로 환산 시 약 0.5일분의 추가 수확 가능성
```

**작물별 비계절 씨앗 가성비**:

| 작물 | 정가 | 비계절 가격 (x2.0) | 추가 비용 | 0.5일 추가 수확 가치 | 순효과 |
|------|------|-------------------|----------|-------------------|--------|
| 감자 (15G) | 15G | 30G | +15G | 2.5G (5.0G/일 x 0.5) | **-12.5G** |
| 당근 (18G) | 18G | 36G | +18G | 2.8G | **-15.2G** |
| 토마토 (30G) | 30G | 60G | +30G | 3.0G | **-27G** |
| 옥수수 (50G) | 50G | 100G | +50G | 3.6G | **-46.4G** |
| 딸기 (40G) | 40G | 80G | +40G | 10.7G | **-29.3G** |
| 호박 (100G) | 100G | 200G | +100G | 5.0G | **-95G** |
| 수박 (175G) | 175G | 350G | +175G | 7.3G | **-167.7G** |

**판정**: 시간 이점만 놓고 보면 비계절 씨앗은 모든 작물에서 **손해**이다. 그러나 비계절 씨앗의 진정한 가치는 "전략적 사전 준비"의 심리적 만족감과, 계절 전환 시 상점 동선을 줄이는 편의성에 있다. 가격 x2.0은 "편의 프리미엄"으로서 과도하다.

**[RISK] 비계절 씨앗의 경제적 가치 부재**: 순수 경제 관점에서 구매 동기가 전혀 없다. 계절 시작 Day 1에 상점에서 정가 구매하면 동일한 결과를 얻을 수 있으므로, x2.0 프리미엄의 정당화 근거가 약하다.

**권장 조정**: 섹션 5.2 참조.

### 3.3 에너지 토닉 (Energy Tonic) -- 200G

**기본 데이터** (-> see `docs/content/npcs.md` 섹션 6.3, 9.3):
- 가격: 200G
- 효과: 에너지 50 즉시 회복
- 제한: 1일 1회
- 등장 확률: 25%

**ROI 분석 -- 에너지의 경제적 가치**:

에너지 50 = 추가 경작 활동 가능량. 에너지 소모 기준 (-> see `docs/systems/farming-system.md` 섹션 3.2):
- 호미 2, 물뿌리개 1, 낫 1

```
에너지 50의 활용 시나리오:
  A) 호미질 25회 -> 25타일 새 경작지 개간 (일회성)
  B) 물주기 50회 -> 50타일 관리 (기존 타일 유지)
  C) 혼합: 호미 10회(20 에너지) + 물주기 30회(30 에너지)
     -> 10타일 추가 개간 + 30타일 물주기

시나리오 C의 경제 가치 (중반, 옥수수 기준):
  추가 10타일 x 옥수수 일일 수익 7.1G = 71G/일 (단, 수확까지 7일 소요)
  즉, 에너지 토닉 1회 투자 -> 7일 후부터 추가 10타일 운영 가능
  
  10타일 x 7.1G/일 x (28-7)일 = 10 x 7.1 x 21 = 1,491G (시즌 잔여분)
  투자: 200G (토닉) + 10타일 씨앗(500G) = 700G
  순이익: 1,491G - 700G = 791G

  -> ROI: 791G / 700G = 113% (시즌 내 회수 가능)
```

그러나 위 시나리오는 "에너지 부족으로 인해 개간할 수 없었던 타일이 있다"는 전제를 요구한다. 에너지가 남는 플레이어에게는 가치가 0이다.

```
에너지 토닉의 실제 가치 판단:
  일일 에너지 100 (-> see `docs/systems/farming-system.md` 섹션 3.2)
  일일 관리 가능 타일: ~25타일

  에너지 토닉 사용 시: 150 에너지 -> ~37타일 관리 가능
  추가 12타일 x 옥수수 일일 수익 7.1G = 85.2G/일의 "잠재 가치"
  
  28일 시즌 기준, 매일 토닉 사용 시:
  - 토닉 비용: 200G x 28 = 5,600G (비현실적 -- 재고 1~3개, 주 2회 구매 기회)
  - 실제 구매 가능량: 시즌당 최대 ~4개 (2주말 x 평균 2개)
  - 실제 투자: 200G x 4 = 800G
  - 추가 수익: 4일 x 85.2G = 340.8G
  
  -> ROI: 340.8G / 800G = 42.6% (낮음)
```

**판정**: 에너지 토닉은 "바쁜 날(대량 파종, 시즌 전환)" 한정으로 의미가 있으나, 일상적 사용에서의 ROI는 낮다. 200G는 중반 플레이어(잔고 ~2,885G)의 약 7%로 적정 부담도 범위 내이다. 다만 실효성 대비 가격이 다소 높다.

**권장 가격**: 유지 (200G). 에너지 토닉은 ROI보다는 "긴급 상황 대비 보험"의 성격이 강하므로, 약간 비싼 가격이 적절하다. 너무 저렴하면 매번 구매하게 되어 에너지 시스템의 의미가 희석된다.

### 3.4 성장 촉진제 (Growth Accelerator) -- 250G

**기본 데이터** (-> see `docs/content/npcs.md` 섹션 6.3, 9.3):
- 가격: 250G
- 효과: 대상 작물 성장 1일 단축 (1회 사용)
- 등장 확률: 15%

**ROI 분석 -- 1일 단축의 경제적 가치**:

성장 1일 단축이 추가 회전을 만들어내는지가 핵심이다.

```
케이스 A: 추가 회전이 발생하는 경우
  호박 10일 -> 9일 (성장 촉진제): floor(28/10)=2회전 -> floor(28/9)=3회전
  추가 1회전 순이익: 200G - 100G = 100G
  촉진제 비용: 250G
  순효과: 100G - 250G = -150G (적자)

  수박 12일 -> 11일: floor(28/12)=2회전 -> floor(28/11)=2회전 (변화 없음)
  수박 12일(주계절 11일) -> 10일: floor(28/10)=2회전 (변화 없음)

케이스 B: 시즌 마지막 수확을 구원하는 경우
  토마토 Day 24 파종: 수확 Day 29 (시즌 초과) -> 촉진제 사용 -> Day 28 (수확 가능!)
  추가 수확 가치: 60G
  순효과: 60G - 250G = -190G (적자)

케이스 C: 수박에 적용 + 주계절 보정 조합
  수박 12일, 주계절 x1.1: ceil(12/1.1) = 11일
  촉진제 -1일: 10일 -> floor(28/10) = 2회전 (기존과 동일)
  수박의 경우 ceil(12/1.1)=11 -> -1 = 10, floor(28/10)=2 vs floor(28/11)=2
  순효과: 0 (변화 없음)
```

**[RISK] 성장 촉진제의 존재 가치 극히 제한적**: 250G를 투자하여 1일 단축해도, 대부분의 경우 시즌 내 추가 회전이 발생하지 않으며, 발생하더라도 추가 순이익이 투자금을 회수하지 못한다. 단일 타일에만 적용되므로 효율이 극도로 낮다.

**권장 조정**: 섹션 5.4 참조.

### 3.5 행운의 부적 (Lucky Charm) -- 400G

**기본 데이터** (-> see `docs/content/npcs.md` 섹션 6.3, 9.3):
- 가격: 400G
- 효과: 하루 동안 Iridium 품질 확률 +5%
- 등장 확률: 10%

**ROI 분석 -- Iridium 확률 증가의 경제적 가치**:

Iridium 품질의 판매가 배수는 x2.0 (-> see `docs/balance/crop-economy.md` 섹션 3). 행운의 부적은 하루 동안 모든 수확물에 Iridium +5% 확률을 부여한다.

```
시나리오: 부적 사용일에 20타일 수확 (옥수수 기준)

기본 상태 (Normal 토양, 비료 없음):
  품질 분포: Normal 67%, Silver 33%, Gold 0%, Iridium 0%
  기대 배수: 0.67 x 1.0 + 0.33 x 1.25 = 1.08

부적 적용 시:
  Iridium +5% -> 기존 확률에서 5%를 Iridium으로 이동
  품질 분포: Normal 62%, Silver 33%, Gold 0%, Iridium 5%
  기대 배수: 0.62 x 1.0 + 0.33 x 1.25 + 0.05 x 2.0 = 0.62 + 0.4125 + 0.10 = 1.1325
  
  증가분: 1.1325 - 1.08 = +0.0525 배수

옥수수 20타일:
  기대 추가 수입: 20 x 100G x 0.0525 = 105G
  부적 비용: 400G
  순효과: 105G - 400G = -295G (적자)

수박 20타일:
  기대 추가 수입: 20 x 350G x 0.0525 = 367.5G
  부적 비용: 400G
  순효과: 367.5G - 400G = -32.5G (근소 적자)

호박 20타일 (가을 보정 x1.1 포함):
  기대 추가 수입: 20 x 200 x 1.1 x 0.0525 = 231G
  순효과: 231G - 400G = -169G (적자)
```

**고품질 조합 (Rich 토양 + 고급 비료)**:

```
Rich(0.4) + 고급 비료(0.3) = 기본점수 0.7
품질 분포: Normal 0%, Silver 0%, Gold 67%, Iridium 33%
기대 배수: 0.67 x 1.5 + 0.33 x 2.0 = 1.005 + 0.66 = 1.665

부적 적용:
Iridium 38% (33% + 5%), Gold 62% (67% - 5%)
기대 배수: 0.62 x 1.5 + 0.38 x 2.0 = 0.93 + 0.76 = 1.69

증가분: 1.69 - 1.665 = +0.025 배수 (Rich 토양에서는 오히려 효과 감소)

수박 20타일: 20 x 350G x 0.025 = 175G
순효과: 175G - 400G = -225G (적자)
```

**[RISK] 행운의 부적은 가격 대비 효과가 현저히 부족**: +5% Iridium 확률의 경제 가치가 400G에 전혀 미치지 못한다. 특히 고품질 토양/비료 조합에서는 효과가 더 작아진다(Iridium이 이미 높으므로 추가 5%의 한계효용 감소). 단일 가장 유리한 조건(수박 20타일)에서조차 근소 적자이다.

**권장 조정**: 섹션 5.5 참조.

### 3.6 정원 등불 (Garden Lantern) -- 300G

**기본 데이터** (-> see `docs/content/npcs.md` 섹션 6.3):
- 가격: 300G
- 효과: 농장 장식 아이템, 야간 조명
- 등장 확률: 15%

**ROI 분석**: 정원 등불은 순수 장식/편의 아이템으로, 직접적인 경제 수익이 없다. 야간 조명 효과는 게임플레이의 시인성을 개선하지만, 게임 내 행동에 경제적 영향을 주지 않는다.

**판정**: 장식 아이템은 ROI가 아닌 "골드 싱크"로서의 역할이 핵심이다. 300G는 중반 플레이어(잔고 ~2,885G)의 ~10%로 적절한 사치 소비이다. 여행 상인의 "특별한 물건" 느낌에 부합하며, 농장 커스터마이징을 원하는 플레이어에게 선택지를 제공한다.

**권장 가격**: 유지 (300G). 장식 아이템의 가격은 ROI 기반이 아닌 "감정 가치" 기반이므로 현재 수준이 적절하다.

### 3.7 풍향계 (Weathervane) -- 500G

**기본 데이터** (-> see `docs/content/npcs.md` 섹션 6.3):
- 가격: 500G
- 효과: 농장 장식 + 다음 날 날씨 예보 기능
- 등장 확률: 10%

**ROI 분석 -- 날씨 예보의 경제적 가치**:

날씨 정보 사전 파악의 가치:
- 비 예보 시 물주기 에너지 절약 (비 올 때 물주기 불필요)
- Storm 다음 날 가격 보정 x1.05 (-> see `docs/balance/crop-economy.md` 섹션 4) 활용 가능

```
물주기 에너지 절약:
  비 확률 ~20% (계절별 상이, -> see `docs/systems/time-season.md`)
  비 예보를 알면 전날 물주기를 건너뛸 수 있다
  에너지 절약: 25타일 x 1 에너지 = 25 에너지
  이 에너지로 추가 경작 가능 -> 25 에너지의 경제 가치: ~85G/일 (에너지 토닉 비례 환산)
  
  연간 비 일수: ~22일 (112일 x 20%)
  날씨 예보 없이도 비 자체는 알 수 있으나, "내일 비가 온다"를 사전에 알면 전날 일과 조정 가능
  -> 실제로 활용 가능한 기회: 연간 ~10회 (비 예보 + 전날 에너지 여유 있을 때)
  
  연간 추가 가치: 10 x 85G = 850G
  회수 기간: 500G / (850G/112일) = ~66일 (약 2.4계절)
```

**판정**: 풍향계는 장식+기능 복합 아이템으로, 연간 약 850G의 간접 가치를 제공한다. 500G 투자는 약 2.4계절(~66일)에 회수되며, 이는 장기 투자로서 합리적이다. 다만 500G는 중반 잔고의 ~17%로 다소 부담이 크다.

**권장 가격**: 유지 (500G). 영구 설치 아이템이므로 높은 초기 투자가 정당화된다. 장식 가치까지 합산하면 적절한 가격이다.

### 3.8 온실 전용 씨앗 (겨울 전용) -- 정가 x1.5

**ROI 분석 -- 겨울 온실 재배 수익**:

이 분석에서 핵심은 "겨울 전용 씨앗을 여행 상인에서만 구매 가능한가"이다. npcs.md [OPEN] 1에서 잡화 상점 판매 여부가 미결이다.

```
시나리오: 여행 상인 독점 판매 (현재 설계)

표고버섯 (가장 효율적인 겨울 작물):

시금치:
```

**권장 가격**: 유지 (정가 x1.5). 겨울 콘텐츠 진입에 적절한 프리미엄이다.

### 4.2 핵심 문제 진단

1. **소비형 버프 아이템(만능 비료, 성장 촉진제, 행운의 부적)의 가성비가 일괄적으로 나쁘다.** 이들은 "1회 사용 후 소멸"이면서 가격이 높아, 투자 대비 회수가 불가능하다. 여행 상인의 골드 싱크 역할과 "비싸지만 가치 있는 선택"이라는 설계 목표가 달성되지 않고 있다.

2. **비계절 씨앗은 경제적 가치가 사실상 0이다.** 계절 시작 Day 1에 상점에서 정가 구매하면 동일 결과를 얻을 수 있어, x2.0 프리미엄의 근거가 없다.

3. **장식/유틸리티 아이템(등불, 풍향계)과 겨울 전용 씨앗은 적정 범위이다.** 이들은 ROI 외의 가치(장식, 편의, 독점 경로)가 가격을 정당화한다.

## 5. 권장 가격 조정

### 5.1 만능 비료 -- 가격 인하: 150G -> 80G

**근거**:
- 현재 150G에서는 어떤 작물에도 흑자를 낼 수 없다
- 고급 비료(50G)와 속성장 비료(40G)의 합계가 90G이므로, "두 가지를 한 번에" 편의성에 대한 프리미엄은 90G 이하가 적절
- 80G = 고급 비료(50G) + 30G 편의 프리미엄 (60% 프리미엄)
- 80G라면 호박 3회전 시 순이익 양전환 가능:
  ```
  호박 3회전(만능 비료 80G): 3 x (200 x 1.21) - 3 x 100 - 3 x 80 = 726 - 300 - 240 = +186G
  비료 없음 호박 2회전: 2 x (200 x 1.08) - 2 x 100 = 432 - 200 = +232G
  ```
  여전히 비료 없음보다 낮지만, 격차가 크게 줄어 "품질+속도 동시 확보"라는 편의 가치를 고려하면 합리적 선택지가 된다
- 조정 범위: (-> see `docs/content/npcs.md` 섹션 9.2) `universalFertPrice` 100~250 범위 내

**[OPEN] 만능 비료의 효과 상향도 검토 필요**: 가격 인하 대신 또는 병행하여 품질 보정을 +15% -> +20%, 성장 속도를 +10% -> +15%로 상향하는 안도 있다. 가격 80G + 효과 상향 시 전문 비료 대비 "편의형 상위 호환"이 되지 않도록 주의 필요.

### 5.2 비계절 씨앗 -- 배수 인하: x2.0 -> x1.5

**근거**:
- 시간 이점의 경제 가치가 거의 0이므로, x2.0 프리미엄은 과도
- x1.5면 겨울 전용 씨앗과 동일한 프리미엄으로 일관성 확보
- 추가 설계 제안: 비계절 씨앗에 "선점 보너스" 부여 -- 해당 씨앗으로 재배한 첫 수확물에 Silver 이상 품질 보장. 이렇게 하면 실질적인 경제 가치가 생기며, "미리 준비하면 좋은 작물이 나온다"는 직관적 보상 구조가 형성된다
- 조정 범위: (-> see `docs/content/npcs.md` 섹션 9.1) `offSeasonSeedPriceMult` 1.5~3.0 범위 내

[OPEN] 비계절 씨앗 "선점 보너스" 도입 여부. 도입 시 작물 품질 시스템(-> see `docs/systems/crop-growth.md`)에 특수 경로 추가가 필요하여 복잡도가 증가한다. 미도입 시 비계절 씨앗의 가치는 순수 편의성에 국한된다.

### 5.3 에너지 토닉 -- 유지: 200G

**근거**:
- 에너지 토닉은 ROI 기반 아이템이 아니라 "보험/긴급 사용" 아이템이다
- 200G는 "고민하게 만드는" 가격(잔고의 ~7%)으로 적절하다
- 더 저렴하면 매일 구매하게 되어 에너지 시스템(100 에너지 제한)의 의미가 희석된다
- 더 비싸면 구매 동기가 완전히 사라진다

### 5.4 성장 촉진제 -- 효과 변경 + 가격 인하: 250G -> 150G, 단축 1일 -> 2일

**근거**:
- 1일 단축은 대부분의 경우 추가 회전을 만들지 못해 실질 가치가 0
- 2일 단축으로 변경하면 유효 케이스가 증가:
  ```
  호박: 10일 -> 8일, floor(28/8)=3회전 (기존 2회전)
  수박: 11일(주계절) -> 9일, floor(28/9)=3회전 (기존 2회전)
  옥수수: 7일 -> 5일, floor(28/5)=5회전 (기존 4회전)
  ```
- 가격 150G에서 수박 추가 1회전 가치: (350 x 1.08) - 175 - 150 = 53G (근소 흑자)
- 조정 범위: (-> see `docs/content/npcs.md` 섹션 9.3) `growthAccelDays` 1~2 범위 내

[OPEN] 성장 촉진제 2일 단축이 비료 효과(속성장 비료: 전체 성장일수 단축)와 중첩 시 과도하게 강해질 수 있다. 촉진제와 비료가 곱연산인지 합연산인지에 따라 밸런스가 달라진다. 상호작용 규칙 확정 필요.

### 5.5 행운의 부적 -- 효과 상향 + 가격 인하: 400G -> 250G, Iridium +5% -> +15%

**근거**:
- Iridium +5%의 경제 가치는 최선의 조건에서도 ~370G에 불과하여 400G 투자를 회수하지 못함
- +15%로 상향 시:
  ```
  Normal 토양 + 비료 없음 (기본점수 0.1):
  부적 없음: Iridium 0%
  부적 적용: Iridium 15%
  기대 배수 증가: +0.15 (Normal에서 Iridium으로 이동 시 +1.0 배수 x 15%)
  
  수박 20타일: 20 x 350G x 0.15 = 1,050G
  부적 비용: 250G
  순효과: +800G (흑자!)
  
  옥수수 20타일: 20 x 100G x 0.15 = 300G
  순효과: 300G - 250G = +50G (근소 흑자)
  ```
- 250G, +15%면 "수확이 많은 날에 사용하면 확실히 이득"이라는 명확한 판단 기준이 생긴다
- 조정 범위: (-> see `docs/content/npcs.md` 섹션 9.3) `luckyCharmIridiumBonus` 0.03~0.10 범위 **초과** -> 범위 확장 필요 (0.03~0.20)

[RISK] Iridium +15%는 Rich 토양 + 고급 비료(기본 Iridium 33%)와 합산 시 48%에 도달한다. 거의 절반이 Iridium이 되어 품질 시스템의 의미가 희석될 수 있다. 상한선으로 "최종 Iridium 확률 50% cap" 규칙을 도입하면 완화 가능하다.

### 6.1 npcs.md [OPEN] 1: 겨울 온실 전용 씨앗 판매 경로

**결정 제안: 여행 상인 독점 유지 + 잡화 상점 겨울 한정 판매 병행**

- 여행 상인에서 x1.5 프리미엄으로 구매 가능 (현행 유지)
- 잡화 상점(하나)에서도 겨울에 한해 정가로 구매 가능하도록 추가
- 단, 잡화 상점은 겨울 2주차(Day 8)부터 판매 시작. 여행 상인은 겨울 1주차부터 판매

**근거**: 
- 여행 상인 독점 시 겨울 콘텐츠 진입이 "주말 방문 + 20% 확률"에 의존하여 불확실성이 과도
- 잡화 상점 병행 시 접근성 보장 + 여행 상인의 가치(1주 선점 + 재고 한정의 희소성)도 유지
- x1.5 프리미엄은 "1주 일찍 시작하는 시간 이점"에 대한 비용으로 정당화

### 6.2 npcs.md [OPEN] 3: 여행 상인 가격 밸런스 (에너지 토닉)

**결정 제안: 에너지 토닉 200G 유지**

- 200G는 에너지 시스템을 "완전히 우회"하기엔 비싸고, "비상 시 사용"하기엔 적절한 가격
- 1일 1회 제한과 결합하여 밸런스 유지

### 6.3 traveler-npc.md [OPEN] 1: 구름이 당근 경로

**결정 제안: 도입 권장**

- 바람이의 친밀도 상승 기회는 주 2일로 제한적 (다른 NPC는 매일 대화 가능)
- 당근 1개(구매가 18G, 판매가 35G) 소비로 친밀도 +1은 합리적 비용
- 구현 복잡도가 낮고(기존 아이템 사용 + 친밀도 +1), 플레이어 경험 개선 효과가 크다
- 주 2회 추가 친밀도 기회로 바람이의 Friend 단계 도달이 합리적 속도가 된다:
  ```
  기존: 주말 방문(대화 +1 x 2일) + 구매(+2 x 평균 1회) = 주당 4pt
  당근 추가: 주당 4pt + 당근 2pt = 주당 6pt
  Friend(40pt) 도달: 기존 10주 -> 당근 포함 ~7주
  ```

### 6.4 traveler-npc.md [OPEN] 3: 연말 특별 세트 밸런스

**결정 제안: 가격 배수 x1.5 유지**

```

  감자 15G x 5 = 75G
  당근 18G x 5 = 90G
  딸기 40G x 5 = 200G
  합계: 365G
  
연말 세트 가격: 365G x 1.5 = 547.5G (548G)
프리미엄: 183G

봄 Day 1 시작 이점:
  15타일 즉시 파종 (감자 5, 당근 5, 딸기 5)
  상점 방문 동선 절약 + 확실한 물량 확보
```

- 1년차 겨울 종료 잔고 ~8,270G 기준 세트 비용은 ~6.6%로 적정
- 봄 시작 이점은 비계절 씨앗과 달리 "묶음 편의성"이 추가되므로 x1.5가 합리적

## 8. Open Questions / Risks

### Open Questions

1. [OPEN] **만능 비료 효과 상향 vs 가격 인하**: 섹션 5.1에서 가격 인하(150G -> 80G)를 제안했으나, 효과 상향(품질 +15% -> +20%, 성장 +10% -> +15%)을 병행할지 검토 필요. 병행 시 전문 비료 대비 상위 호환 우려.

2. [OPEN] **비계절 씨앗 "선점 보너스" 도입 여부**: 비계절 씨앗에 Silver 이상 품질 보장 효과를 추가할지. 도입 시 품질 시스템 특수 경로 추가 필요.

3. [OPEN] **성장 촉진제 x 비료 상호작용 규칙**: 촉진제 2일 단축과 속성장 비료의 중첩 시 곱연산/합연산 여부 확정 필요.

4. [OPEN] **행운의 부적 Iridium +15% 상한 규칙**: Rich 토양 + 고급 비료 + 부적 조합 시 Iridium 48% 도달. 50% cap 도입 여부.

5. [OPEN] **미끼(Bait) 시스템과의 연계**: fishing-system.md [OPEN] 4에서 미끼 시스템 도입이 미결. 미끼가 도입되면 "고급 낚싯대 미끼"를 여행 상인 아이템 풀에 추가하는 것이 자연스럽다. 미끼 미도입 시 현재 아이템 풀 8종을 유지한다.

6. [OPEN] **npcs.md 섹션 9.3 `luckyCharmIridiumBonus` 조정 범위 확장**: 현재 0.03~0.10인데, 권장 값 0.15를 수용하려면 상한을 0.20으로 확장해야 한다.

### Risks

1. [RISK] **소비형 아이템 가격 일괄 인하 시 골드 싱크 약화**: 만능 비료(150->80G), 성장 촉진제(250->150G), 행운의 부적(400->250G)을 모두 인하하면, 여행 상인의 골드 싱크 역할이 크게 축소된다. 대안으로 고가 장식 아이템(500G+)을 추가 투입하여 보완할 수 있다.

2. [RISK] **행운의 부적 +15% Iridium이 품질 시스템을 왜곡할 가능성**: 특히 수확일에 부적을 사용하면 대량의 Iridium 품질 작물이 쏟아져, 경제 시스템(수급 변동)에 일시적 과잉 공급이 발생할 수 있다.

3. [RISK] **성장 촉진제 2일 단축의 체감 밸런스**: 수박에서 3회전이 가능해지면 시즌 수익이 급등한다(175G x 3 = 525G vs 350G). 단일 타일 한정이므로 대규모 영향은 없으나, 고레벨 플레이어가 촉진제를 대량 비축하면 문제가 될 수 있다 (재고 1~3개 제한이 완화 역할).

4. [RISK] **겨울 씨앗 이중 판매 경로(여행 상인 + 잡화 상점)가 여행 상인의 가치를 희석할 가능성**: 섹션 6.1의 제안대로 잡화 상점이 겨울 2주차부터 판매하면, 여행 상인의 "1주 선점" 가치가 약해질 수 있다. 잡화 상점 판매 시작을 겨울 3주차(Day 15)로 늦추는 것을 대안으로 검토.
