> 작성: Claude Code (Opus) | 2026-04-07

## 1. Context

이 문서는 SeedMind의 세 가지 XP 소스(수확/경작, 퀘스트, 업적)를 통합 관점에서 분석하고, 단일 XP 예산 체계를 확립한다. 각 소스가 독립 설계되어 합산 시 필요 XP의 347%에 달하는 인플레이션 문제(-> see `docs/balance/quest-rewards.md` 섹션 3.4)를 해결하기 위한 재설계 문서이다.

### 1.1 본 문서가 canonical인 데이터

- XP 소스별 비율 목표 (연간 예산 배분)
- 통합 시나리오 시뮬레이션 결과 및 권장안
- 1년차 레벨 목표 확정 값
- XP 소스별 연간 상한(hard cap / soft target)
- progression-curve.md 수정 필요 사항 목록

### 1.2 본 문서가 canonical이 아닌 데이터 (참조만)

| 데이터 종류 | 참조처 |
|------------|--------|
| XP 테이블 (baseXP, growthFactor), 레벨별 누적 XP | `docs/balance/progression-curve.md` 섹션 1.3.2 (원본) 및 섹션 2.4.1 (조정) |
| 수확/경작 XP 시뮬레이션 (1년차 ~3,332 XP) | `docs/balance/progression-curve.md` 섹션 2.4.2 |
| 퀘스트 카테고리별 XP 집계 및 조정 제안 A | `docs/balance/quest-rewards.md` 섹션 6.2 |
| 업적 30종 전체 XP (2,250 XP) | `docs/content/achievements.md` 섹션 2.4 |
| 업적 카테고리별 XP 소계 | `docs/content/achievements.md` 섹션 3~9 |

### 1.3 선행 분석의 불일치 식별

[RISK] **BAL-006에서 두 가지 XP 테이블을 혼용하고 있다.** BAL-006 섹션 3.1은 원본 테이블(baseXP=50, growthFactor=1.55, 레벨 10 = 4,609 XP)을 기준으로 퀘스트 XP 비율을 분석하나, 섹션 7.2의 수확/경작 XP(~3,332 XP)는 조정된 테이블(baseXP=80, growthFactor=1.60, 레벨 10 = 9,029 XP)의 시뮬레이션 결과이다. 이 혼용으로 인해 BAL-006의 "레벨 9 후반 도달" 판정이 부정확하다.

**본 문서는 조정된 XP 테이블(섹션 2.4.1)을 canonical 기준으로 채택한다.** 이유: progression-curve.md가 이미 물주기 XP 제거, baseXP/growthFactor 상향 등의 조정을 확정 반영했으므로, 원본 테이블은 역사적 참조에 불과하다.

## 2. 설계 목표 재정의

### 2.1 1년차 레벨 목표

현행 설계(-> see `docs/balance/progression-curve.md` 섹션 2.4.3): 수확/경작 XP만으로 1년차 레벨 7 직전(~3,332 XP, 레벨 7 = 2,104 XP 초과, 레벨 8 = 3,446 XP 미달).

퀘스트와 업적 XP를 포함한 통합 레벨 목표:

| 플레이어 유형 | 1년차 목표 레벨 | 근거 |
|-------------|---------------|------|
| 캐주얼 (퀘스트 미수행, 업적 자연 달성만) | 7 | 수확/경작만으로 레벨 7 도달은 이미 검증됨 |
| 일반 (퀘스트 50% 수행, 업적 중간 진행) | **8** | 레벨 8 = 3,446 XP. 수확 3,332 + 보조 소스 ~500~800으로 도달 가능해야 함 |
| 적극적 (퀘스트 대부분 수행, 업적 적극 추적) | **9 초반** | 레벨 9 = 5,593 XP. 수확 3,332 + 보조 소스 ~1,500~2,000으로 근접 |

**확정 목표**: 1년차 종료 시 **일반 플레이어 레벨 8, 적극적 플레이어 레벨 9 초반**. 레벨 10(9,029 XP)은 2년차 중후반 달성 콘텐츠로 유지한다.

이전 설계의 "1년차 레벨 7~8"(-> see `docs/balance/progression-curve.md` 섹션 2.4.3)과 비교하면, 퀘스트/업적 보조 소스를 포함하여 **일반 플레이어 레벨 8**로 상향 조정한다. 이는 퀘스트를 수행하는 플레이어에게 의미 있는 진행 보상을 주면서도, 퀘스트를 안 하는 캐주얼 플레이어도 레벨 7에서 핵심 콘텐츠(수박, 가공소)를 모두 해금할 수 있게 한다.

### 2.2 XP 소스별 역할 정의

| XP 소스 | 역할 | 성격 |
|---------|------|------|
| **수확/경작** | Primary (주 소스) | 게임의 핵심 루프. 매일 반복하는 행위에서 꾸준히 획득. 레벨업의 기반 |
| **퀘스트** | Secondary (보조 가속) | 특정 목표 달성 시 보너스. 퀘스트를 수행하면 레벨업이 약간 빨라지되, 안 해도 치명적이지 않음 |
| **업적** | Tertiary (장기 보너스) | 마일스톤 달성 시 일회성 보상. 대부분 자연스럽게 달성되며, 의도적 추적은 소수 |

### 2.3 XP 소스별 비율 목표

**1년차 기준 XP 비율 (일반 플레이어)**:

| XP 소스 | 비율 목표 | XP 범위 (레벨 8 = 3,446 기준) | 근거 |
|---------|----------|---------------------------|------|
| 수확/경작 | **75~80%** | 2,585~2,757 XP | 핵심 루프가 레벨업의 대부분을 책임 |
| 퀘스트 | **12~15%** | 414~517 XP | 보조 가속, 퀘스트 미수행 시 1레벨 차이 |
| 업적 | **5~10%** | 172~345 XP | 자연 달성분만 반영, 의도적 추적 불필요 |
| **합계** | **~100%** | ~3,446 XP | 레벨 8 도달 |

**적극적 플레이어 기준 (레벨 9 초반 = ~4,500 XP 목표)**:

| XP 소스 | 예상 비율 | 예상 XP |
|---------|----------|---------|
| 수확/경작 | ~74% | ~3,332 XP (시뮬레이션 값 그대로) |
| 퀘스트 | ~18% | ~800 XP |
| 업적 | ~8% | ~370 XP |
| **합계** | ~100% | ~4,502 XP |

[OPEN] 수확/경작 ~3,332 XP는 progression-curve.md의 "일반 플레이어" 시뮬레이션이다. 적극적 플레이어는 최적화 재배로 ~4,000 XP까지 가능하므로(-> see `docs/balance/progression-curve.md` 섹션 2.5), 적극적 플레이어의 수확/경작 비율이 더 높아질 수 있다. 이 경우 레벨 9 중반까지 도달 가능하나, 레벨 10과의 간격이 충분하므로 허용 범위 내이다.

### 3.3 시나리오 B: 퀘스트 XP 삭감 + XP 테이블 소폭 하향

퀘스트 XP는 제안 A를 적용하되, growthFactor를 1.55로 되돌려 후반 레벨 문턱을 낮춘다. baseXP=80은 유지하여 초반 가속 억제는 보존.

**수정된 XP 테이블** (baseXP=80, growthFactor=1.55):

| 레벨 | 필요 XP | 누적 XP |
|------|---------|---------|
| 1 | - | 0 |
| 2 | 80 | 80 |
| 3 | 124 | 204 |
| 4 | 192 | 396 |
| 5 | 298 | 694 |
| 6 | 462 | 1,156 |
| 7 | 716 | 1,872 |
| 8 | 1,110 | 2,982 |
| 9 | 1,720 | 4,702 |
| 10 | 2,666 | 7,368 |

검산: `필요_XP(n) = round(80 * 1.55^(n-1))`
- 레벨 2: round(80 * 1) = 80
- 레벨 3: round(80 * 1.55) = 124
- 레벨 4: round(80 * 2.4025) = 192
- 레벨 5: round(80 * 3.724) = 298
- 레벨 6: round(80 * 5.772) = 462
- 레벨 7: round(80 * 8.946) = 716
- 레벨 8: round(80 * 13.867) = 1,110 (오차 수정: round = 1,109, 반올림 차이 허용)
- 레벨 9: round(80 * 21.494) = 1,720
- 레벨 10: round(80 * 33.316) = 2,665 (반올림 차이 허용)

**1년차 통합 시뮬레이션**:

| XP 소스 | 일반 플레이어 | 적극적 플레이어 |
|---------|-------------|---------------|
| 수확/경작 | ~3,332 XP | ~3,332 XP |
| 퀘스트 | ~400 XP | ~550 XP |
| 업적 | ~500 XP | ~660 XP |
| **합계** | **~4,232 XP** | **~4,542 XP** |

**레벨 도달 판정**:

| 플레이어 | 누적 XP | 도달 레벨 | 비고 |
|---------|---------|----------|------|
| 캐주얼 (수확만) | ~3,332 XP | 8 (2,982 초과, 4,702 미달) | 수확만으로 레벨 8 도달 -- **과도** |
| 일반 | ~4,232 XP | 8 (4,702 미달) | 레벨 9 근접 |
| 적극적 | ~4,542 XP | 8 (4,702 미달, 160 XP 부족) | 레벨 9 직전 |

**판정**: 캐주얼 플레이어가 수확/경작만으로 레벨 8에 도달하여, 퀘스트/업적의 보조 역할이 무의미해진다. "퀘스트를 하면 1레벨 더 빨리" 라는 인센티브가 사라진다. **탈락**.

[RISK] growthFactor를 낮추면 후반 레벨 간격이 좁아져, 수확 XP만으로 충분한 레벨업이 가능해지며 보조 소스의 존재 가치가 희석된다.

#### 4.2.1 보완: 퀘스트 XP 소폭 상향 (제안 A 수정)

BAL-006 제안 A의 퀘스트 XP 총량을 692에서 **900 XP**로 상향한다. 이는 전체 XP(9,029)의 약 10%이며, 1년차 실현분은 약 700 XP가 된다.

**수정 근거**: 제안 A(692 XP)는 퀘스트 XP를 지나치게 낮춰 "퀘스트를 해도 레벨업에 거의 기여하지 않는다"는 체감을 줄 수 있다. 900 XP로 상향하면:
- 전체 XP(9,029)의 ~10%로, 설계 목표(10~15%) 하한 부합
- 1년차 실현(~740 XP)은 레벨 8->9 필요 XP(2,147)의 약 34% -- 의미 있는 기여이나 지배적이지 않음

**레벨 도달 (gF=1.60 테이블 기준)**:

| 플레이어 | 누적 XP | 도달 레벨 | 다음 레벨까지 | 비고 |

| XP 소스 | 제한 방식 | 상세 |
|---------|----------|------|

### 5.4 연간 XP 수지 안전 검증

레벨 10(9,029 XP) 도달 시점 검증:

```
최적 시나리오 (적극적 플레이어):
  연간 수확/경작: ~4,000 XP
  연간 퀘스트: 1년차 ~740, 2년차 ~160 (잔여분)
  연간 업적: 1년차 ~660, 2년차 ~800, 3년차+ ~790

  1년차 말: ~5,400 XP (레벨 8 후반)
  2년차 말: ~5,400 + 4,000 + 160 + 800 = ~10,360 XP -> 레벨 10 돌파 (2년차 중후반)

최악 시나리오 (캐주얼 플레이어, 퀘스트/업적 최소):
  연간 수확/경작: ~3,332 XP
  연간 퀘스트: ~0 XP
  연간 업적: ~200 XP (자연 달성)

  1년차 말: ~3,532 XP (레벨 8 갓 진입)
  2년차 말: ~3,532 + 3,332 + 200 = ~7,064 XP (레벨 9, 5,593 초과)
  3년차 중: ~7,064 + 2,000 = ~9,064 XP -> 레벨 10 돌파 (3년차 중반)
```

**판정**: 적극적 플레이어 2년차 중후반, 캐주얼 플레이어 3년차 중반에 레벨 10 도달. 실시간으로 적극적 ~35시간, 캐주얼 ~50시간. 게임 장르 특성상 적절한 범위이다.

- [OPEN] **일일 목표 레벨 스케일링 제거 여부**: 제안 A에서 일일 목표의 레벨 스케일링 제거를 권장했는데, 이를 유지하되 상한만 낮추는(x1.0~x1.4) 방안이 플레이어 체감상 나을 수 있다. quest-system.md에서 결정 필요.

## Risks

- [RISK] **BAL-006 기준 XP 불일치**: BAL-006가 원본 테이블(4,609)과 조정 테이블(9,029)을 혼용하고 있어, BAL-006의 일부 판정이 부정확하다. BAL-006 수정이 선행되어야 본 문서의 제안이 정확히 구현된다.
- [RISK] **업적 XP 장기 누적**: 전 업적 완료 시 2,250 XP는 레벨 10(9,029)의 25%. 이는 3년 이상에 걸쳐 분산되므로 연 단위 인플레이션은 제한적이나, 만약 플레이어가 업적을 집중 공략하면 특정 시점에 XP 급등이 발생할 수 있다. 단계형 업적의 Gold 단계(100 XP)가 집중 해금될 때 모니터링 필요.
- [RISK] **캐주얼 플레이어의 레벨 8 진입**: 수확/경작만으로 ~3,532 XP에 도달하면 레벨 8(3,446)에 걸치는데, 이 수치는 "일반 플레이어" 시뮬레이션 기준이다. 비효율적 플레이어는 ~2,500~3,000 XP에 머물 수 있으며, 이 경우 레벨 7에 머문다. 이는 오히려 의도에 부합한다.
- [RISK] **제안 A'의 퀘스트 XP 900은 quest-system.md의 기존 목표(10~15%)의 상한(10%)에 맞춘 것이다**. 만약 후속 콘텐츠(DLC, 시즌 이벤트 등)에서 퀘스트가 추가되면 이 예산이 초과될 수 있다. 확장 콘텐츠의 XP 예산은 별도 관리해야 한다.
