## Context

이 문서는 SeedMind 1년차(봄~겨울 28일 × 4계절 = 112일) 플레이어의 5개 수익 소스(작물/낚시/채집/가공/목축) 합산 총수익 범위를 검증한다. 초기 자본(시작 골드) 대비 경제 성장 곡선이 설계 의도에 부합하는지 확인하고, 소스 간 비중 목표가 유지되는지 검증한다.

**본 문서가 canonical인 데이터**: 시나리오별 연간 총수익 시뮬레이션 결과, 소스 비중 분석, 설계 목표 달성 여부 판정.

**원시 수치의 canonical 출처**:

### 3.1 시나리오 A — 초보자 (전업 작물 재배)

**플레이어 프로필**: 초보 플레이어. 작물 재배에 집중, 낚시 미활용, 채집 최소(Zone D 봄~가을 1회 순회), 가공소 미해금, 목축 미시작.

**경작 규모**: 12타일 (초보자 기준, 에너지 여유 감안)

#### 봄 (Day 1~28) — 작물 수익

초보자는 감자/당근부터 시작, 딸기는 Lv.4 해금이므로 첫 봄에는 미접근.

```
감자 9타일: 9 x 135G = 1,215G
당근 3타일: 3 x 153G = 459G
봄 작물 총 순이익: 1,215G + 459G = 1,674G
씨앗 총 비용: 9 x 15G x 9회전 + 3 x 18G x 9회전
           = 1,215G(씨앗) + 486G(씨앗) = 1,701G
주의: 순이익은 이미 씨앗비 차감 후 기준 (-> see docs/balance/crop-economy.md 섹션 1.2)
실제 봄 수입(판매가): 9 x 9 x 30G + 3 x 9 x 35G = 2,430G + 945G = 3,375G
봄 순이익 = 3,375G - 1,701G = 1,674G (일치)
```

채집 (Zone D, Lv.1, 봄, 수급 미적용 격일 순회): 88G/일 × 28일 = 2,464G
단, 초보자는 채집 최소화 가정. 격일 순회 실질 수입:
  - 리스폰 일(~14회): 88G × 14 ≈ 1,232G

**봄 총수익 (수입 합산)**:
```
작물 판매 수입: 3,375G
씨앗 비용:    -1,701G
채집 수입:    +1,232G
봄 순수익:   = 2,906G
```

#### 여름 (Day 29~56) — 작물 수익

초보자는 토마토(Lv.2) + 옥수수(Lv.3) 위주. 수박(Lv.7)은 미해금.

```
토마토 6타일: 6 x 150G = 900G
옥수수 6타일: 6 x 200G = 1,200G
여름 순이익: 2,100G
여름 작물 판매 수입: 6 x 5회전 x 60G + 6 x 4회전 x 100G
                 = 1,800G + 2,400G = 4,200G
씨앗 비용: 6 x 5 x 30G + 6 x 4 x 50G = 900G + 1,200G = 2,100G
순이익 검증: 4,200G - 2,100G = 2,100G (일치)
```

채집 여름: 88G(봄) x 1.14 ≈ Zone D 여름 100G/일 (-> see `docs/balance/gathering-economy.md` 섹션 8.1). 격일 14회: 100G × 14 = 1,400G

**여름 총수익**:
```
작물 수입:   4,200G
씨앗 비용:  -2,100G
채집 수입:  +1,400G
여름 순수익: = 3,500G
```

#### 가을 (Day 57~84) — 작물 수익

초보자, Lv.5 도달 가정. 호박(Lv.5) 재배 시작 가능.

```
당근 6타일: 6 x 153G = 918G
호박 6타일: 가을 보정 포함 6 x 273G = 1,638G (-> see docs/balance/crop-economy.md 섹션 4.1)

가을 작물 판매 수입:
  당근 6타일 x 9회전 x 35G x 1.1(가을보정) = 6 x 9 x 38.5G = 2,079G
  호박 6타일 x 2회전: 1차(Day10, x1.1) + 2차(Day20, x1.265) = (220+253)G x 6타일 = 2,838G
  합계: 4,917G
씨앗 비용:
  당근: 6 x 9 x 18G = 972G
  호박: 6 x 2 x 100G = 1,200G
  합계: 2,172G
가을 순이익: 4,917G - 2,172G = 2,745G
```

채집 가을: Zone D Lv.1 가을 119G/일 (수급 미적용, 격일). 격일 14회: 119G × 14 = 1,666G

**가을 총수익**:
```
작물 수입:   4,917G
씨앗 비용:  -2,172G
채집 수입:  +1,666G
가을 순수익: = 4,411G
```

#### 겨울 (Day 85~112) — 온실 집중

온실 미보유(Lv.5 미달 또는 2,000G 미투자) 시나리오 A:
- 야외 작물 재배 불가, 시설 건설에 집중
- 채집: 10G/일 × 28일 = 280G
- 낚시: 가벼운 얼음 낚시(~5마리/일). 빙어 18G x 5 x 0.95 ≈ 86G/일 × 28일 = 2,408G

```
겨울 수입:
  채집: 280G
  낚시: 2,408G
  겨울 수익: 2,688G
```

**핵심 관찰**:

1. **작물 비중 미달 원인**: 낚시의 에너지 효율이 작물 대비 지나치게 높다. 수급 적용 후에도 Lv.5 낚시 474G/일은 옥수수 24칸(171G/일)의 2.8배. "경작 후 남은 에너지 전부 낚시" 전략이 최적화되면 작물 비중이 구조적으로 억제된다.

2. **가공이 핵심 수익원**: 시나리오 B에서 가공 부가가치 52,660G는 작물 단독(23,150G)의 2.3배. "가공 체인 2~3x ROI" 목표는 달성되었으나, 가공이 작물을 포함하는 개념으로 보면 (작물+가공) 합산 시 75,810G = 56%. 이 관점에서는 목표에 근접.

3. **낚시 보조 수입 초과**: 전 시나리오에서 낚시가 작물 수익을 상회하거나 동등. "보조 수입" 포지셔닝 재검토 필요.

### 4.3 이상 징후 및 개선 제안

#### 이상 징후 1: 낚시 수익 과도

**현상**: 시나리오 B에서 낚시(46,624G) > 작물(22,772G). 낚시 집중 전략이 경작보다 수익성 높음.

**원인**:
- 씨앗 비용 없음 (투입 비용 = 에너지만)
- 수급 적용 후에도 일당 315~474G/일로 작물 대비 높음
- BAL-013 성공률 하향 적용 후에도 Lv.5+ 에서 경작 능가

**제안**: 낚시는 이미 BAL-013에서 조정됨. 추가 조정보다는 작물 경작의 경제적 가치 상향(비료 효율 개선 등)이 근본적 해결책.

[OPEN] 작물이 "경제 핵심"이 되려면 작물 순이익의 실질 상향 또는 낚시 에너지 소모 증가가 필요. 현재 구조에서 작물 50~70% 목표 달성이 어려울 수 있음.

#### 이상 징후 2: 온실 딸기의 압도적 겨울 지배

**현상**: 겨울 온실 딸기 16타일 = 9,600G 순이익. 겨울 전용 작물(표고버섯 5,920G)을 크게 상회.

**원인**: 딸기의 구조적 다중 수확 효율이 온실 환경에서도 유지됨 (-> see `docs/balance/crop-economy.md` 섹션 4.2 [RISK]).

**제안**: 온실 딸기의 재수확 간격 상향(3일 → 4일) 또는 온실 내 다중 수확 제한. [OPEN] BAL-001에서 이미 식별된 이슈 — 딸기 재수확 조정이 선행되어야 함.

#### 이상 징후 3: 초보자 경제 진입 장벽

**현상**: 시나리오 A에서 봄 순수익 2,906G. 여름 씨앗 구매(예: 수박 10개 = 1,750G)와 가공소 건설(예상 ~2,000G) 사이에서 자원 배분 압박이 발생.

**판정**: 의도된 설계. "500G 시작으로 선택이 필요하다"는 설계 목표에 부합. 1년차 말 13,500G는 충분한 성장으로, 2년차 수박+온실 전략 전환이 자연스럽게 가능.

## 5. 열린 문제 및 리스크

### [OPEN] OPEN#1: 작물 비중 50~70% 목표 현실성

낚시의 구조적 효율로 인해 작물 비중 50~70% 목표 달성이 어렵다. "작물+가공"을 하나의 경제 소스로 묶어 50~70%로 정의하는 방식으로 목표를 재정의하는 것이 현실적.

### [OPEN] OPEN#2: 치즈 공방 연료/에너지 비용

시나리오 C에서 치즈 공방 연료비를 0G로 가정했으나 미확정. livestock-system.md 섹션 7.2 확정 후 재계산 필요.

### [OPEN] OPEN#3: 온실 딸기 vs 겨울 전용 작물 균형

온실 딸기가 겨울 전용 작물을 압도. BAL-001 딸기 재수확 조정 결정 전까지 겨울 경제 밸런스 미확정.

### [RISK] 낚시 우위 구조 지속

BAL-013 조정 후에도 Lv.5+ 낚시가 동일 에너지의 경작 대비 수익 2배 이상. "경작이 핵심" 포지셔닝 위협. 장기적 재조정 검토 필요.

### [RISK] 가공 체인의 과도한 수익 집중

시나리오 C에서 베이커리 호박 파이 가공이 단일 레시피로 거대한 수익(잠재 124,600G/시즌)을 생성. 재료 병목이 유일한 제약. 호박 보관 기간 제한 도입을 검토해야 함 (-> see `docs/balance/processing-economy.md` 섹션 2.7 [RISK]).
