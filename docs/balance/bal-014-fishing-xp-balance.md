## 1. 개요

BAL-013에서 미니게임 성공률을 하향 조정(Lv.1: 50%, Lv.5: 65%, Lv.10: 80%)한 결과, 낚시 숙련도 XP 획득 속도에 직접적인 영향이 발생한다. 본 문서는 변경된 성공률 기반으로 **낚시 숙련도 레벨 10 도달 일수**를 재시뮬레이션하고, 기존 추산치(약 45일)의 유효성을 검증한다.

**핵심 질문**: BAL-013 성공률 하향 후에도 숙련도 Lv.10 도달이 1.5~2계절(42~56일) 범위 내에서 달성 가능한가?

## 3. BAL-013 조정이 XP에 미치는 영향

### 3.1 성공률 변경 요약

(-> see `docs/systems/fishing-system.md` 섹션 7.4)

| 숙련도 | BAL-013 확정 성공률 |
|--------|-------------------|
| Lv.1 | 50% |
| Lv.5 | 65% |
| Lv.10 | 80% |

### 3.2 XP 획득에 대한 영향 경로

성공률 하향은 두 가지 경로로 XP 획득에 영향을 미친다.

**경로 A — 포획 XP 감소**: 성공률이 낮으면 포획 횟수가 줄어 포획 XP(5~50 XP)가 감소한다.

**경로 B — 실패 XP 증가**: 실패 시에도 1 XP를 획득하므로, 실패가 많아지면 실패 XP가 보전 역할을 한다.

**경로 C — 에너지 효율 악화**: 실패 시 에너지 3 소모(성공 시 2)이므로, 성공률 하향은 1회당 평균 에너지를 올려 총 시도 횟수를 줄인다.

### 3.3 1회 시도당 기대 XP 산출

일반 낚시 시 포획 어종의 기대 XP:
```
포획 기대 XP = Common(0.70) * 5 + Uncommon(0.25) * 10 + Rare(0.05) * 20
            = 3.5 + 2.5 + 1.0 = 7.0 XP/포획
```

보물 상자 보정 (Lv.1 기준 3% 확률, 8 XP):
```
보물 기대 XP = 0.03 * 8 = 0.24 XP/포획
```

**1회 시도당 기대 XP** (성공률 반영):

| 숙련도 | 성공률 | 성공 XP | 실패 XP | 기대 XP/시도 |
|--------|--------|---------|---------|-------------|
| Lv.1 | 50% | 0.50 * (7.0 + 0.24) = 3.62 | 0.50 * 1 = 0.50 | **4.12 XP** |
| Lv.3 | ~55% (보간) | 0.55 * 7.24 = 3.98 | 0.45 * 1 = 0.45 | **4.43 XP** |
| Lv.5 | 65% | 0.65 * 7.24 = 4.71 | 0.35 * 1 = 0.35 | **5.06 XP** |
| Lv.7 | ~73% (보간) | 0.73 * 7.24 = 5.29 | 0.27 * 1 = 0.27 | **5.56 XP** |
| Lv.8 | ~76% (보간) | 0.76 * 7.24 = 5.50 | 0.24 * 1 = 0.24 | **5.74 XP** |

**참고**: Lv.4 이후 희귀종 확률 보정(x1.1, x1.2)이 적용되나, 본 시뮬레이션에서는 Rare 비율 5%→5.5~6% 수준의 미미한 변화이므로 간략화를 위해 기본 분포로 계산한다. 이는 보수적 추정에 해당한다.

## 5. 밸런스 판정 및 확정 수치

### 5.1 기존 추산과의 괴리 분석

기존 추산(45일)은 "하루 10회 낚시, Common 중심(5 XP/회)"으로 단순 계산한 것이다:
```
2,250 XP / (10회 * 5 XP) = 45일
```

이 추산의 문제점:
1. **실패 XP 미반영**: 미니게임 실패 시 1 XP가 누적되는 것을 무시함
2. **에너지 기반 시도 횟수 미반영**: Lv.8 에너지 반감 효과로 시도 횟수가 급증하는 것을 미반영
3. **희귀도 혼합 미반영**: Uncommon(10 XP)과 Rare(20 XP)의 혼합 효과 무시
4. **성공률 변화 미반영**: 숙련도가 오르면 성공률이 50%→80%로 상승하여 포획 비율 증가

### 5.2 밸런스 판정

**문제**: 시나리오 A(21일)는 목표 범위(1.5~2계절 = 42~56일)의 절반 수준이다. 시나리오 C(13일)는 2주 만에 마스터에 도달하여 숙련도 성장 체감이 너무 빠르다.

**원인**: Lv.8 에너지 반감(캐스팅 2→1)이 XP 획득 속도를 과도하게 가속한다.

**조정안 2 (대안): 행동당 XP 하향**

XP 요구량은 유지하되, 포획 XP를 하향한다 (예: Common 3 XP, Uncommon 6 XP, Rare 12 XP, Legendary 30 XP).

**비채택 사유**: 포획당 XP가 너무 적으면 1회 낚시의 성장 피드백이 약해져 "보상감"이 저하된다. XP 요구량 상향이 체감적으로 더 자연스럽다.

### 5.4 최종 확정

**조정안 1을 채택한다.** 근거:

1. 시나리오 A(일반 플레이) 기준 **39일(약 1.4계절)**은 목표 범위(1.5~2계절) 하단에 근접하여 적절하다
2. 시나리오 B(캐주얼) 기준 **68일(약 2.4계절)**은 캐주얼 플레이어에게도 달성 가능한 범위이다
3. 시나리오 C(최적화) 기준 **24일(약 0.9계절)**은 전념 플레이어에게 적절한 보상이다
4. Lv.간 XP가 100씩 균등 증가하는 단순한 구조로 직관적이다

[OPEN] 시나리오 A의 39일이 목표 하한(42일)에 약간 못 미친다. Lv.8 에너지 반감 효과 때문인데, 이를 추가 상향하면 캐주얼 플레이어(68일)의 부담이 커진다. 39일은 수용 가능 범위로 판단한다.

## 8. Open Questions / Risks

1. [OPEN] Lv.8 에너지 반감(캐스팅 2→1)이 XP 가속의 핵심 원인이다. 에너지 반감을 Lv.9로 미루거나, 반감 대신 1.5로 조정하는 대안도 고려할 수 있으나, 이는 수익 밸런스(fishing-economy.md)에도 영향을 미치므로 별도 분석이 필요하다.

2. [OPEN] 조정안 1 채택 시 fishing-system.md 섹션 7.2의 해금/보상 조건(예: Lv.3에서 강화 낚싯대 해금)은 변경 없이 유지한다. XP 요구량만 상향되므로 해금 시점이 늦어지는 효과가 있다. 강화 낚싯대 해금이 Lv.3(누적 300 XP)이므로 시나리오 A 기준 약 5일(기존 2~3일)로 늦어진다. 이 지연이 초반 체감에 부정적이지 않은지 플레이테스트에서 검증 필요.

3. [RISK] 보물 상자 XP(8 XP)와 Giant 변이 XP(x2)는 본 시뮬레이션에서 최소 반영되었다. 실제 플레이에서는 이들이 누적되어 시뮬레이션보다 5~10% 빠른 도달이 예상된다.

4. [RISK] 일일 퀘스트("오늘 물고기 5마리를 잡으세요")의 낚시 XP 보너스(-> see `docs/systems/fishing-system.md` 섹션 8.3)가 시뮬레이션에 미포함이다. 퀘스트 보너스 XP가 크면 도달 일수가 추가 단축될 수 있다.
