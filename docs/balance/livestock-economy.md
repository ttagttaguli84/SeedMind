### 1.2 닭 (Chicken) ROI 분석

**기본 수치** (-> see `docs/content/livestock-system.md` 섹션 1.1, 4.1):
- 구매가: 800G
- 일일 사료비: 10G
- 생산물: 달걀(35G), 매일 생산
- 고품질: 대형 달걀(70G), 행복도 200 시 25% 확률 (-> see `docs/content/livestock-system.md` 섹션 4.2)

```
일일 순수익 (일반) = 35G - 10G = 25G/일
일일 순수익 (기대값, 행복도 200) = (35G * 0.75 + 70G * 0.25) - 10G = (26.25 + 17.5) - 10 = 33.75G/일
```

| 지표 | 일반 품질 | 기대값 (행복도 200) |
|------|----------|-------------------|
| 일일 순수익 | 25G | 33.75G |
| Break-even 일수 | 800 / 25 = **32일** | 800 / 33.75 = **24일** |
| 30일 순수익 | 25 * 30 - 800 = **-50G** | 33.75 * 30 - 800 = **212.5G** |
| 60일 순수익 | 25 * 60 - 800 = **700G** | 33.75 * 60 - 800 = **1,225G** |
| ROI (60일) | 700 / 800 = **87.5%** | 1,225 / 800 = **153%** |

**ROI 등급: B** (안정적이나 단가 낮음. 대량 사육 시 효율 향상)

**작물 비교**: 감자의 일일 수익률 5.0G/일 (-> see `docs/balance/crop-economy.md` 섹션 1.2) 대비 닭은 25G/일로 5배. 단, 감자는 초기 투자 15G(씨앗)인 반면 닭은 800G. Break-even까지 32일이 필요하여, 장기 투자 성격이 분명하다.

### 1.3 염소 (Goat) ROI 분석

**기본 수치** (-> see `docs/content/livestock-system.md` 섹션 1.1, 4.1):
- 구매가: 2,000G
- 일일 사료비: 20G
- 생산물: 염소젖(80G), 2일 주기
- 고품질: 고급 염소젖(160G), 행복도 200 시 20% 확률 (-> see `docs/content/livestock-system.md` 섹션 4.2)

```
일일 순수익 (일반) = (80G / 2일) - 20G = 40 - 20 = 20G/일
일일 순수익 (기대값, 행복도 200) = ((80 * 0.8 + 160 * 0.2) / 2) - 20 = (64 + 32) / 2 - 20 = 48 - 20 = 28G/일
```

| 지표 | 일반 품질 | 기대값 (행복도 200) |
|------|----------|-------------------|
| 일일 순수익 | 20G | 28G |
| Break-even 일수 | 2,000 / 20 = **100일** | 2,000 / 28 = **72일** |
| 30일 순수익 | 20 * 30 - 2,000 = **-1,400G** | 28 * 30 - 2,000 = **-1,160G** |
| 60일 순수익 | 20 * 60 - 2,000 = **-800G** | 28 * 60 - 2,000 = **-320G** |
| ROI (60일) | -800 / 2,000 = **-40%** | -320 / 2,000 = **-16%** |

**ROI 등급: C** (직판 단독으로는 회수 기간이 지나치게 길다. 치즈 공방 가공 연계가 필수)

[RISK] 염소는 60일 시점에서도 투자를 회수하지 못한다. 이는 의도적 설계(치즈 공방 연계 전제)인지, 밸런스 문제인지 명확히 해야 한다. 섹션 5에서 조정안을 제시한다.

### 1.4 소 (Cow) ROI 분석

**기본 수치** (-> see `docs/content/livestock-system.md` 섹션 1.1, 4.1):
- 구매가: 4,000G
- 일일 사료비: 30G
- 생산물: 우유(120G), 2일 주기
- 고품질: 고급 우유(240G), 행복도 200 시 20% 확률 (-> see `docs/content/livestock-system.md` 섹션 4.2)

```
일일 순수익 (일반) = (120G / 2일) - 30G = 60 - 30 = 30G/일
일일 순수익 (기대값, 행복도 200) = ((120 * 0.8 + 240 * 0.2) / 2) - 30 = (96 + 48) / 2 - 30 = 72 - 30 = 42G/일
```

| 지표 | 일반 품질 | 기대값 (행복도 200) |
|------|----------|-------------------|
| 일일 순수익 | 30G | 42G |
| Break-even 일수 | 4,000 / 30 = **134일** | 4,000 / 42 = **96일** |
| 30일 순수익 | 30 * 30 - 4,000 = **-3,100G** | 42 * 30 - 4,000 = **-2,740G** |
| 60일 순수익 | 30 * 60 - 4,000 = **-2,200G** | 42 * 60 - 4,000 = **-1,480G** |
| ROI (60일) | -2,200 / 4,000 = **-55%** | -1,480 / 4,000 = **-37%** |

**ROI 등급: C** (직판 단독으로 회수 매우 느림. 치즈 공방 가공이 사실상 필수)

### 1.5 양 (Sheep) ROI 분석

**기본 수치** (-> see `docs/content/livestock-system.md` 섹션 1.1, 4.1):
- 구매가: 3,000G
- 일일 사료비: 25G
- 생산물: 양모(150G), 3일 주기
- 고품질: 고급 양모(300G), 행복도 200 시 15% 확률 (-> see `docs/content/livestock-system.md` 섹션 4.2)

```
일일 순수익 (일반) = (150G / 3일) - 25G = 50 - 25 = 25G/일
일일 순수익 (기대값, 행복도 200) = ((150 * 0.85 + 300 * 0.15) / 3) - 25 = (127.5 + 45) / 3 - 25 = 57.5 - 25 = 32.5G/일
```

| 지표 | 일반 품질 | 기대값 (행복도 200) |
|------|----------|-------------------|
| 일일 순수익 | 25G | 32.5G |
| Break-even 일수 | 3,000 / 25 = **120일** | 3,000 / 32.5 = **93일** |
| 30일 순수익 | 25 * 30 - 3,000 = **-2,250G** | 32.5 * 30 - 3,000 = **-2,025G** |
| 60일 순수익 | 25 * 60 - 3,000 = **-1,500G** | 32.5 * 60 - 3,000 = **-1,050G** |
| ROI (60일) | -1,500 / 3,000 = **-50%** | -1,050 / 3,000 = **-35%** |

**ROI 등급: C** (양모의 높은 단가에도 불구하고, 3일 주기와 높은 사료비가 수익률을 억제)

### 1.6 동물별 ROI 비교 종합

| 동물 | 일일 순수익 (일반) | Break-even (일반) | Break-even (기대값) | ROI 등급 |
|------|-----------------|------------------|-------------------|---------|
| 닭 | 25G | 32일 | 24일 | **B** |
| 염소 | 20G | 100일 | 72일 | C |
| 소 | 30G | 134일 | 96일 | C |
| 양 | 25G | 120일 | 93일 | C |

**핵심 발견**:
1. **닭이 압도적으로 효율적**: Break-even 32일은 다른 동물(100~134일)의 1/3~1/4 수준. 초기 투자 대비 가장 빠른 회수.
2. **중대형 동물은 직판만으로 부족**: 염소/소/양은 60일 시점에서도 투자 미회수. 가공 체인 연계 없이는 경제적 정당성이 약하다.
3. **일일 순수익은 소가 최고**: 소(30G/일) > 닭=양(25G/일) > 염소(20G/일). 장기적으로 소의 절대 수익이 가장 크다.

## 2. 가공 체인 ROI 분석

### 2.1 예상 가공품 부가가치

치즈 공방 가공 레시피는 미확정이나, 방향성이 제시되어 있다 (-> see `docs/content/livestock-system.md` 섹션 7.2).

| 원재료 | 예상 가공품 | 원재료 판매가 | 기대 배율 | 예상 가공품 가격 | 예상 처리 시간 |
|--------|-----------|------------|----------|---------------|-------------|
| 우유 (120G) | 치즈 | 120G | ~1.8x | ~216G | [OPEN] |
| 우유 (120G) | 버터 | 120G | ~1.5x | ~180G | [OPEN] |
| 염소젖 (80G) | 염소 치즈 | 80G | ~2.0x | ~160G | [OPEN] |
| 양모 (150G) | 직물 | 150G | [OPEN] | [OPEN] | [OPEN] |

> 기대 배율은 livestock-system.md 섹션 7.2의 예비 수치. 확정 레시피와 처리 시간은 치즈 공방 문서(미작성)가 canonical이 될 예정.

[OPEN] 양모 가공(직물)은 livestock-system.md에 기대 배율이 명시되어 있지 않다. 기존 가공 시스템의 일반 가공소 배율(~1.5x~2.0x, -> see `docs/content/processing-system.md` 섹션 3)을 참고하여 ~1.6x를 잠정 제안한다.

### 2.2 가공 후 ROI 재계산 (소)

치즈 가공 기준 (예상):

```
우유 직판 일일 순수익: 30G/일 (섹션 1.4)
치즈 가공 일일 순수익: (216G / 2일) - 30G = 108 - 30 = 78G/일
가공 부가가치: 78 - 30 = +48G/일 (+160%)
```

| 지표 | 직판 | 치즈 가공 (예상) |
|------|------|----------------|
| 일일 순수익 | 30G | 78G |
| Break-even (동물만) | 134일 | 4,000 / 78 = **52일** |
| 60일 순수익 | -2,200G | 78 * 60 - 4,000 = **+680G** |

가공 연계 시 소의 ROI 등급: **C → A** (52일 Break-even은 닭 직판(32일)보다 길지만, 일일 절대 수익이 월등)

### 2.3 가공 후 ROI 재계산 (염소)

염소 치즈 가공 기준 (예상):

```
염소젖 직판 일일 순수익: 20G/일 (섹션 1.3)
염소 치즈 가공 일일 순수익: (160G / 2일) - 20G = 80 - 20 = 60G/일
가공 부가가치: 60 - 20 = +40G/일 (+200%)
```

| 지표 | 직판 | 염소 치즈 가공 (예상) |
|------|------|-------------------|
| 일일 순수익 | 20G | 60G |
| Break-even (동물만) | 100일 | 2,000 / 60 = **34일** |
| 60일 순수익 | -800G | 60 * 60 - 2,000 = **+1,600G** |

가공 연계 시 염소의 ROI 등급: **C → A** (가장 극적인 수익 개선. 치즈 공방 가공이 염소 경제성의 핵심)

### 2.4 가공 체인 ROI 종합

| 동물 | 직판 일일 순수익 | 가공 일일 순수익 (예상) | 가공 Break-even | 가공 ROI 등급 |
|------|---------------|---------------------|---------------|-------------|
| 닭 | 25G | - (가공 대상 아님) | 32일 (직판) | B |
| 염소 | 20G | 60G | 34일 | **A** |
| 소 | 30G | 78G | 52일 | **A** |
| 양 | 25G | ~40G (직물 잠정) | ~75일 | B |

**핵심 발견**:
1. **치즈 공방은 목축 경제의 핵심 전환점**: 가공 없이는 닭만 경제적이나, 가공 연계 시 염소/소가 최고 효율 동물로 부상
2. **염소의 반전**: 직판 최하위(C)에서 가공 최상위(A)로. 치즈 공방 건설의 가장 강력한 동기 부여
3. **양은 가공 체인이 약함**: 직물 가공의 부가가치가 불명확하여, 양모 직판에 의존하는 구조. 가공 연계 설계가 추가로 필요

[OPEN] 양모 → 직물 가공 경로의 구체적 배율과 처리 시간이 미확정. 양의 경제적 정당성을 확보하려면 직물 배율을 ~1.8x 이상으로 설정하거나, 양모 직판가를 상향해야 할 수 있다.

### 3.3 투자 타이밍 분석

**레벨 6 도달 시 예상 잔고**: 약 2,000~3,000G (-> see `docs/systems/economy-system.md` 섹션 5.2, 중반 가을 시작 자금 ~2,000G)

| 목축 시나리오 A | 7,100G | 50G | 142일 | 재투자 불필요 (사료만) |
| 목축 시나리오 B | 12,700G | 130G | 98일 | 재투자 불필요 |
| 목축 시나리오 D (가공 포함) | 17,200G | 258G | 67일 | 가공 슬롯 점유 |

> 작물 ROI 참조: 옥수수 순이익 50G/개, 성장일수 7일 (-> see `docs/balance/crop-economy.md` 섹션 1.2). 20타일 기준 시즌당 3~4회전.

**핵심 차이점**:
- **작물**: 낮은 초기 투자, 빠른 회전, 매 시즌 씨앗 재구매. 에너지/시간 집약적.
- **목축**: 높은 초기 투자, 느린 회수, 영구적 수입원. 일일 돌봄 에너지만 소모(4 에너지/마리, -> see `docs/content/livestock-system.md` 섹션 2.5).
- **병행 최적**: 작물로 현금 흐름 확보 → 여유 자금으로 목축 투자 → 안정적 수입원 축적의 2단계 전략이 최적.

## 4. 계절별 수익 시뮬레이션

### 4.1 시나리오 B: 중간 투자 (닭x4 + 소x1) 장기 시뮬레이션

| 기간 | 일일 수입 | 일일 사료비 | 일일 순수익 | 누적 순수익 | 투자 대비 |
|------|----------|-----------|-----------|-----------|----------|
| 1~28일 (1계절) | 200G | 70G | 130G | 3,640G | -9,060G |
| 29~56일 (2계절) | 200G | 70G | 130G | 7,280G | -5,420G |
| 57~84일 (3계절) | 200G | 70G | 130G | 10,920G | -1,780G |
| 85~98일 (3.5계절) | 200G | 70G | 130G | 12,740G | +40G |
| 85~112일 (4계절) | 200G | 70G | 130G | 14,560G | +1,860G |

> 일일 수입 계산: 달걀 x4 = 140G + 우유(2일 평균) 60G = 200G/일 (-> see `docs/content/livestock-system.md` 섹션 8.2)

**겨울 시 참고**: 생산물은 계절에 관계없이 생산된다 (-> see `docs/content/livestock-system.md` 섹션 5). 단, 겨울철 행복도 감소(-5/일, 외양간 Lv.2 히터로 해소 가능)가 생산 배수에 영향을 줄 수 있다.

### 4.2 고품질 생산물 포함 기대값 시뮬레이션

행복도 200 유지 가정 시 고품질 확률 반영 (-> see `docs/content/livestock-system.md` 섹션 4.2):

| 동물 | 일반 수입 | 고품질 기대 수입 | 증가율 |
|------|----------|---------------|--------|
| 닭 x4 | 140G/일 | (35*0.75 + 70*0.25)*4 = 175G/일 | +25% |
| 소 x1 (2일 평균) | 60G/일 | (120*0.8 + 240*0.2)/2 = 72G/일 | +20% |
| **합계** | **200G/일** | **247G/일** | **+23.5%** |

고품질 포함 시 시나리오 B Break-even: 12,700 / (247 - 70) = **72일** (~2.6계절). 일반 대비 26일 단축.

## 5. 밸런스 평가 및 조정 제안

### 5.1 현재 밸런스의 문제점

| # | 문제 | 심각도 | 근거 |
|---|------|--------|------|
| 1 | **염소 직판 ROI 과도하게 낮음** | 높음 | Break-even 100일. 닭(32일)의 3배 이상. 치즈 공방 없이 염소를 구매할 경제적 동기가 없다 |
| 2 | **소 직판 ROI 과도하게 낮음** | 중간 | Break-even 134일이나, 일일 순수익(30G)은 최고이므로 장기 투자로 정당화 가능 |
| 3 | **양의 가공 경로 부재** | 중간 | 양모 가공(직물)의 배율/처리시간이 미정. 양의 장기 경제성이 불투명 |
| 4 | **닭 편중 가능성** | 높음 | 닭만 8마리 사육하면 Break-even 24일(기대값), 일일 350G. 다른 동물 구매 동기 약화 |
| 5 | **초기 진입 비용 과다** | 중간 | 레벨 6 잔고(~2,000~3,000G)로 최소 투자(7,100G)도 즉시 불가 |

### 5.2 조정 제안

#### 제안 1: 염소 사료비 하향 (20G → 15G)

```
조정 후 일일 순수익: (80 / 2) - 15 = 25G/일 (기존 20G → +25%)
조정 후 Break-even: 2,000 / 25 = 80일 (기존 100일 → 20% 단축)
```

[OPEN] 이 조정은 `docs/content/livestock-system.md` 섹션 1.1의 canonical 수치 변경을 수반한다. 채택 시 해당 문서를 먼저 수정해야 한다.

#### 제안 2: 닭장 최대 수용 수 감소 (Lv.1: 4→3, Lv.2: 8→6)

닭 편중을 방지하기 위해 닭장 수용 수를 줄여, 중대형 동물 구매를 유도한다.

[OPEN] 이 조정은 `docs/content/livestock-system.md` 섹션 3.1의 canonical 수치 변경을 수반한다. 닭장 수용 수 변경 시 Zone E 타일 배분(섹션 6.2)도 재검토 필요.

#### 제안 3: 치즈 공방 해금 조건 완화 (레벨 8 → 레벨 7)

치즈 공방을 1레벨 일찍 해금하여, 염소/소 구매와 가공 연계를 빠르게 실현할 수 있게 한다. 현재 레벨 8 해금은 중대형 동물의 긴 Break-even과 맞물려 목축 경험을 지연시킨다.

[OPEN] 치즈 공방 해금 레벨은 `docs/content/livestock-system.md` 섹션 7.1에 예비 값(레벨 8)으로 기재. 변경 시 progression-curve.md 레벨 테이블(섹션 1.3.2)도 갱신 필요.

#### 제안 4: Zone E 해금 비용 또는 닭장 건설 비용 소폭 하향

초기 진입 장벽 완화를 위해 닭장 Lv.1 비용을 1,500G → 1,200G로 하향하면, 최소 투자가 7,100G → 6,800G로 소폭 감소한다.

[OPEN] 이 조정의 효과는 미미(300G)하므로, Zone E 해금 비용(4,000G) 자체를 3,500G로 하향하는 것이 더 효과적일 수 있다. 다만 farm-expansion.md의 Zone 비용 체계 전체에 영향.

## 8. Open Questions / Risks

### Open Questions

1. [OPEN] **치즈 공방 레시피 확정**: 가공품 가격, 처리 시간이 미확정이므로 본 문서의 가공 ROI(섹션 2)는 추정치. 치즈 공방 문서 작성 시 본 문서를 갱신해야 한다 (섹션 2.1~2.4)
2. [OPEN] **양모 가공 경로**: 양모 → 직물 가공의 배율/처리시간 미정. 양의 경제적 위치가 불투명 (섹션 2.4)
3. [OPEN] **염소 사료비 조정 여부**: 20G → 15G 하향 시 canonical 문서 변경 필요 (섹션 5.2 제안 1)
4. [OPEN] **닭장 수용 수 조정 여부**: 닭 편중 방지를 위한 수용 수 감소 (섹션 5.2 제안 2)
5. [OPEN] **치즈 공방 해금 레벨**: 레벨 8 → 7 완화 여부 (섹션 5.2 제안 3)
6. [OPEN] **초기 진입 비용 조정**: Zone E 또는 닭장 비용 하향 여부 (섹션 5.2 제안 4)
7. [OPEN] **수급 변동 적용 여부**: 동물 생산물에도 대량 판매 시 가격 하락이 적용되는지. 적용 시 고정 수입 모델이 무너짐 (-> see `docs/systems/economy-system.md` 섹션 2.6)

### Risks

1. [RISK] **닭 편중 메타**: 닭 8마리 최적화가 다른 동물 구매를 억제할 수 있다. 닭장 수용 제한 또는 닭 판매가 조정이 필요할 수 있다 (섹션 5.1 문제 4)
2. [RISK] **가공 의존도**: 염소/소의 경제성이 치즈 공방에 과도하게 의존. 치즈 공방 건설 전까지의 "빈 구간"에서 플레이어 좌절 가능 (섹션 5.1 문제 1~2)
3. [RISK] **목축 XP 가속**: 중간 투자 시 1계절 1,064 XP는 레벨 6→7 필요 XP(839, -> see `docs/balance/progression-curve.md` 섹션 2.4.1)의 약 1.3배. 작물 XP 합산 시 후반 레벨링이 과도하게 빨라질 수 있다 (-> see `docs/balance/progression-curve.md` 섹션 1.2.6 [OPEN])
4. [RISK] **가공 ROI 추정치 의존**: 본 문서의 가공 분석(섹션 2)은 미확정 배율에 기반. 실제 치즈 공방 수치 확정 시 대폭 변경 가능
