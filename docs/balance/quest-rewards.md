- 퀘스트 카테고리별 골드/XP 보상 집계 및 비율 분석
- 퀘스트 보상이 전체 경제/XP 진행에 미치는 영향 수치

- 밸런스 판정 및 조정 제안

**canonical 출처 참조**:

## 3. XP 기여도 분석

### 3.1 기준 XP 데이터

> **[수정 — BAL-007]** 이하 섹션 3.1~3.3은 BAL-006 작성 당시의 구 XP 수치 분석을 역사 기록으로 보존한다. BAL-006은 구 XP 테이블(baseXP=50, growthFactor=1.55, 레벨 10 = 4,609 XP)을 기준으로 작성되었으나, canonical XP 테이블은 이미 조정 후 값(baseXP=80, growthFactor=1.60, 레벨 10 = 9,029 XP)으로 확정되었다. 최종 확정값은 BAL-007을 참조한다 (-> see `docs/balance/xp-integration.md`).

전체 누적 XP (레벨 10 도달): **9,029 XP** — canonical 확정값 (-> see `docs/balance/progression-curve.md` 섹션 2.4.1)  
*(구 분석 기준: 4,609 XP — 섹션 1.3.2, 현재 [DEPRECATED])*

퀘스트 XP 확정: 전체의 약 **10% = ~900 XP** (BAL-007 제안 A' 확정, -> see `docs/balance/xp-integration.md`)  
*(구 분석 기준: 전체의 10~15% = 461~692 XP)*

### 3.3 판정 (BAL-007 이후 업데이트)

**[RESOLVED — BAL-007]** 구 수치 분석에서 제기된 XP 인플레이션 문제는 BAL-007에서 해소되었다.

BAL-006 당시 판정: 퀘스트 XP 9,147 XP = 구 기준(4,609 XP)의 198.5% 초과 → 인플레이션 RISK

통합 시뮬레이션 결과는 `docs/balance/xp-integration.md` 참조.

## 5. 일일 목표 레벨별 수익 분석

### 5.1 레벨별 일일 목표 예상 골드/XP

> **[DEPRECATED — XP 열 폐기, 골드 열 유지]** 아래 표의 XP 수치(17.9/21.5/25.1/30.4/34.0 XP)는 BAL-007 A' 이전 구 XP 기반값이다. BAL-007 A' 적용 후 일일 목표 XP는 1~3 XP(레벨 스케일링 전)로 대폭 삭감되어 해당 XP 열은 유효하지 않다. 골드 분석은 유효하다.

레벨 스케일링: `기본_보상 * (1 + (level - 1) * 0.1)`

일일 2개 완료, 50% 완료율 기준 = 평균 1개/일 완료

- [OPEN] 농장 도전(fc_)의 XP를 대폭 삭감하면 장기 플레이 동기가 약화될 수 있다. 도전 보상을 XP 대신 고유 아이템/칭호로 대체하는 방안 검토 필요.

## Risks

- [RISK] **XP 인플레이션**: 현행 수치 그대로 구현 시, 퀘스트 XP만으로 레벨 10 도달 가능. 진행 곡선 전체가 무의미해진다. 우선 해결 필요.
- [RISK] **골드 인플레이션**: 1년차 퀘스트 골드가 작물 수익을 능가하여, 농장 경영이 아닌 퀘스트 수행이 주 수입원이 되는 역전 현상.
- [RISK] **겨울 밸런스 붕괴**: 겨울 메인 퀘스트 1,500G는 작물 수익(2,000G)의 75%로, 경제적 긴장감이 사라진다.
- [RISK] **소형 농장 도전 수치 미확정**: 22개 중 15개의 정확한 보상이 quest-system.md에 상세 기재되어 있지 않아, 본 분석의 추정치(평균 200G + 100 XP)가 실제와 다를 수 있다.
